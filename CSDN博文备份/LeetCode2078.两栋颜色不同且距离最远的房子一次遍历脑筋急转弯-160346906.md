<h2><a id="LetMeFly2078_1"></a>【LetMeFly】2078.两栋颜色不同且距离最远的房子：一次遍历(脑筋急转弯)</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/two-furthest-houses-with-different-colors/" rel="nofollow">https://leetcode.cn/problems/two-furthest-houses-with-different-colors/</a></p> <br><p>街上有 <code>n</code> 栋房子整齐地排成一列，每栋房子都粉刷上了漂亮的颜色。给你一个下标从 <strong>0</strong> 开始且长度为 <code>n</code> 的整数数组 <code>colors</code> ，其中 <code>colors[i]</code> 表示第  <code>i</code> 栋房子的颜色。</p> <br><p>返回 <strong>两栋</strong> 颜色 <strong>不同</strong> 房子之间的 <strong>最大</strong> 距离。</p> <br><p>第 <code>i</code> 栋房子和第 <code>j</code> 栋房子之间的距离是 <code>abs(i - j)</code> ，其中 <code>abs(x)</code> 是 <code>x</code> 的绝对值。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/0c5aa07242f3215ec0618a73e9d533bb.png" /></p> <br><pre><strong>输入：</strong>colors = [<strong><em>1</em></strong>,1,1,<em><strong>6</strong></em>,1,1,1]<br><strong>输出：</strong>3<br><strong>解释：</strong>上图中，颜色 1 标识成蓝色，颜色 6 标识成红色。<br>两栋颜色不同且距离最远的房子是房子 0 和房子 3 。<br>房子 0 的颜色是颜色 1 ，房子 3 的颜色是颜色 6 。两栋房子之间的距离是 abs(0 - 3) = 3 。<br>注意，房子 3 和房子 6 也可以产生最佳答案。<br></pre> <br><p><strong>示例 2：</strong></p> <br><p><img alt="" [<em><strong>1</strong></em>,8,3,8,<em><strong>3</strong></em>]<br><strong>输出：</strong>4<br><strong>解释：</strong>上图中，颜色 1 标识成蓝色，颜色 8 标识成黄色，颜色 3 标识成绿色。<br>两栋颜色不同且距离最远的房子是房子 0 和房子 4 。<br>房子 0 的颜色是颜色 1 ，房子 4 的颜色是颜色 3 。两栋房子之间的距离是 abs(0 - 4) = 4 。<br></pre> <br><p><strong>示例 3：</strong></p> <br><pre><strong>输入：</strong>colors = [<em><strong>0</strong></em>,<em><strong>1</strong></em>]<br><strong>输出：</strong>1<br><strong>解释：</strong>两栋颜色不同且距离最远的房子是房子 0 和房子 1 。<br>房子 0 的颜色是颜色 0 ，房子 1 的颜色是颜色 1 。两栋房子之间的距离是 abs(0 - 1) = 1 。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>n == colors.length</code></li><li><code>2 <= n <= 100</code></li><li><code>0 <= colors[i] <= 100</code></li><li>生成的测试数据满足 <strong>至少 </strong>存在 2 栋颜色不同的房子</li></ul> <br><h3><a id="_60"></a>解题方法：一次遍历</h3> <br><p>距离一个房子最远的异色房子，这个最远怎么找？当然是想优先寻找最左或最有的房子。</p> <br><p>诶，那如果第一个房子和最后一个房子颜色不同，是不是就可以直接返回<span
