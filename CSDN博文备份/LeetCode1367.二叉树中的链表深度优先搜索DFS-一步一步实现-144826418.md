<h2><a id="LetMeFly1367DFS___1"></a>【LetMeFly】1367.二叉树中的链表：深度优先搜索(DFS) - 一步一步实现</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/linked-list-in-binary-tree/" rel="nofollow">https://leetcode.cn/problems/linked-list-in-binary-tree/</a></p> <br><p>给你一棵以 <code>root</code> 为根的二叉树和一个 <code>head</code> 为第一个节点的链表。</p> <br><p>如果在二叉树中，存在一条一直向下的路径，且每个点的数值恰好一一对应以 <code>head</code> 为首的链表中每个节点的值，那么请你返回 <code>True</code> ，否则返回 <code>False</code> 。</p> <br><p>一直向下的路径的意思是：从树中某个节点开始，一直连续向下的路径。</p> <br><p> </p> [4,2,8], root = [1,4,4,null,2,2,null,1,null,6,8,null,null,null,null,1,3]<br><strong>输出：</strong>true<br><strong>解释：</strong>树中蓝色的节点构成了与链表对应的子路径。<br></pre> <br><p><strong>示例 2：</strong></p> <br><p><strong><img alt="" src="https://i-blog.csdnimg.cn/img_convert/e226cd9c649ab381dcd129a56edc076f.png" /></strong></p> <br><pre><strong>输入：</strong>head = [1,4,2,6], root = [1,4,4,null,2,2,null,1,null,6,8,null,null,null,null,1,3]<br><strong>输出：</strong>true<br></pre> <br><p><strong>示例 3：</strong></p> <br><pre><strong>输入：</strong>head = [1,4,2,6,8], root = [1,4,4,null,2,2,null,1,null,6,8,null,null,null,null,1,3]<br><strong>输出：</strong>false<br><strong>解释：</strong>二叉树中不存在一一对应链表的路径。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li>二叉树和链表中的每个节点的值都满足 <code>1 <= node.val <= 100</code> 。</li><li>链表包含的节点数目在 <code>1</code> 到 <code>100</code> 之间。</li><li>二叉树包含的节点数目在 <code>1</code> 到 <code>2500</code> 之间。</li></ul>
