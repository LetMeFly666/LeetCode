<h2><a id="LetMeFly3218_IDFS_1"></a>【LetMeFly】3218.切蛋糕的最小总开销 I：记忆化搜索(深度优先搜索DFS)</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/minimum-cost-for-cutting-cake-i/" rel="nofollow">https://leetcode.cn/problems/minimum-cost-for-cutting-cake-i/</a></p> <br><p>有一个 <code>m x n</code> 大小的矩形蛋糕，需要切成 <code>1 x 1</code> 的小块。</p> <br><p>给你整数 <code>m</code> ，<code>n</code> 和两个数组：</p> <br><ul><li><code>horizontalCut</code> 的大小为 <code>m - 1</code> ，其中 <code>horizontalCut[i]</code> 表示沿着水平线 <code>i</code> 切蛋糕的开销。</li><li><code>verticalCut</code> 的大小为 <code>n - 1</code> ，其中 <code>verticalCut[j]</code> 表示沿着垂直线 <code>j</code> 切蛋糕的开销。</li></ul> <br><p>一次操作中，你可以选择任意不是 <code>1 x 1</code> 大小的矩形蛋糕并执行以下操作之一：</p> <br><ol><li>沿着水平线 <code>i</code> 切开蛋糕，开销为 <code>horizontalCut[i]</code> 。</li><li>沿着垂直线 <code>j</code> 切开蛋糕，开销为 <code>verticalCut[j]</code> 。</li></ol> <br><p>每次操作后，这块蛋糕都被切成两个独立的小蛋糕。</p> <br><p>每次操作的开销都为最开始对应切割线的开销，并且不会改变。</p> <br><p>请你返回将蛋糕全部切成 <code>1 x 1</code> 的蛋糕块的 <strong>最小</strong> 总开销。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>m = 3, n = 2, horizontalCut = [1,3], verticalCut = [5]</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>13</span></p> <br><p><strong>解释：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/cd2e8c3709161b9c2e7a89ffc06d5299.gif" /></p> <br><ul><li>沿着垂直线 0 切开蛋糕，开销为 5 。</li><li>沿着水平线 0 切开 <code>3 x 1</code> 的蛋糕块，开销为 1 。</li><li>沿着水平线 0 切开 <code>3 x 1</code> 的蛋糕块，开销为 1 。</li><li>沿着水平线 1 切开 <code>2 x 1</code> 的蛋糕块，开销为 3 。</li><li>沿着水平线 1 切开 <code>2 x 1</code> 的蛋糕块，开销为 3 。</li></ul> <br><p>总开销为 <code>5 + 1 + 1 + 3 + 3 = 13</code> 。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>m = 2, n = 2, horizontalCut = [7], verticalCut = [4]</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>15</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>沿着水平线 0 切开蛋糕，开销为 7 。</li><li>沿着垂直线 0 切开 <code>1 x 2</code> 的蛋糕块，开销为 4 。</li><li>沿着垂直线 0 切开 <code>1 x 2</code> 的蛋糕块，开销为 4 。</li></ul> <br><p>总开销为 <code>7 + 4 + 4 = 15</code> 。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= m, n <= 20</code></li><li><code>horizontalCut.length == m - 1</code></li><li><code>verticalCut.length == n - 1</code></li><li><code>1 <= horizontalCut[i], verticalCut[i] <= 10<sup>3</sup></code></li></ul> <br><h3><a id="_82"></a>解题方法：深度优先搜索（记忆化）</h3> <br><p>写一个函数<code>dfs</code>计算：切割竖直方向<code>[ia,
