<h2><a id="LetMeFly944Python_1"></a>【LetMeFly】944.删列造序：字符串数组遍历（Python一行版）</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/delete-columns-to-make-sorted/" rel="nofollow">https://leetcode.cn/problems/delete-columns-to-make-sorted/</a></p> <br><p>给你由 <code>n</code> 个小写字母字符串组成的数组 <code>strs</code>，其中每个字符串长度相等。</p> <br><p>这些字符串可以每个一行，排成一个网格。例如，<code>strs = ["abc", "bce", "cae"]</code> 可以排列为：</p> <br><pre>abc<br>bce<br>cae</pre> <br><p>你需要找出并删除 <strong>不是按字典序非严格递增排列的</strong> 列。在上面的例子（下标从 0 开始）中，列 0（<code>'a'</code>, <code>'b'</code>, <code>'c'</code>）和列 2（<code>'c'</code>, <code>'e'</code>, <code>'e'</code>）都是按字典序非严格递增排列的，而列 1（<code>'b'</code>, <code>'c'</code>, <code>'a'</code>）不是，所以要删除列 1 。</p> <br><p>返回你需要删除的列数。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><strong>输入：</strong>strs = ["cba","daf","ghi"]<br><strong>输出：</strong>1<br><strong>解释：</strong>网格示意如下：<br>  cba<br>  daf<br>  ghi<br>列 0 和列 2 按升序排列，但列 1 不是，所以只需要删除列 1 。<br></pre> <br><p><strong>示例 2：</strong></p> <br><pre><strong>输入：</strong>strs = ["a","b"]<br><strong>输出：</strong>0<br><strong>解释：</strong>网格示意如下：<br>  a<br>  b<br>只有列 0 这一列，且已经按升序排列，所以不用删除任何列。<br></pre> <br><p><strong>示例 3：</strong></p> <br><pre><strong>输入：</strong>strs = ["zyx","wvu","tsr"]<br><strong>输出：</strong>3<br><strong>解释：</strong>网格示意如下：<br>  zyx<br>  wvu<br>  tsr<br>所有 3 列都是非升序排列的，所以都要删除。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>n == strs.length</code></li><li><code>1 <= n <= 100</code></li><li><code>1 <= strs[i].length <= 1000</code></li><li><code>strs[i]</code> 由小写英文字母组成</li></ul> <br><h3><a id="_68"></a>解题方法：遍历</h3>
