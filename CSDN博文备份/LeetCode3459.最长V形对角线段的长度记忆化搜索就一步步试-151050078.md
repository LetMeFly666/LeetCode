<h2><a id="LetMeFly3459_V__1"></a>【LetMeFly】3459.最长 V 形对角线段的长度：记忆化搜索——就一步步试</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/length-of-longest-v-shaped-diagonal-segment/" rel="nofollow">https://leetcode.cn/problems/length-of-longest-v-shaped-diagonal-segment/</a></p> <br><p>给你一个大小为 <code>n x m</code> 的二维整数矩阵 <code>grid</code>，其中每个元素的值为 <code>0</code>、<code>1</code> 或 <code>2</code>。</p> <br><p><strong>V 形对角线段</strong> 定义如下：</p> <br><ul><li>线段从 <code>1</code> 开始。</li><li>后续元素按照以下无限序列的模式排列：<code>2, 0, 2, 0, ...</code>。</li><li>该线段： <br>  <ul><li>起始于某个对角方向（左上到右下、右下到左上、右上到左下或左下到右上）。</li><li>沿着相同的对角方向继续，保持 <strong>序列模式 </strong>。</li><li>在保持 <strong>序列模式 </strong>的前提下，最多允许 <strong>一次顺时针 90 度转向 </strong>另一个对角方向。</li></ul> </li></ul> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/862b0368b3865cc038bf4e14fd77b7ad.jpeg" /></p> <br><p>返回最长的 <strong>V 形对角线段 </strong>的 <strong>长度 </strong>。如果不存在有效的线段，则返回 0。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">grid = src="https://i-blog.csdnimg.cn/img_convert/a069f13356398e9732ffac9846cf9bfc.jpeg" /></p> <br><p>最长的 V 形对角线段长度为 5，路径如下：<code>(0,2) → (1,3) → (2,4)</code>，在 <code>(2,4)</code> 处进行 <strong>顺时针 90 度转向 </strong>，继续路径为 <code>(3,3) → (4,2)</code>。</p> <br><p><strong>示例 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">grid = [[2,2,2,2,2],[2,0,2,2,0],[2,0,1,1,0],[1,0,2,2,2],[2,0,0,2,2]]</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">4</span></p> <br><p><strong>解释：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/c328450c78e0513f55ce27f207e47d00.jpeg" /></p> <br><p>最长的 V 形对角线段长度为 4，路径如下：<code>(2,3) → (3,2)</code>，在 <code>(3,2)</code> 处进行 <strong>顺时针 90 度转向 </strong>，继续路径为 <code>(2,1) → (1,0)</code>。</p> <br><p><strong>示例 3：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">grid = [[1,2,2,2,2],[2,2,2,2,0],[2,0,0,0,0],[0,0,2,2,2],[2,0,0,2,0]]</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">5</span></p> <br><p><strong>解释：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/a4ec86b4cc907637afdfca5761d55ef3.jpeg" /></p> <br><p>最长的 V 形对角线段长度为 5，路径如下：<code>(0,0) → (1,1) → (2,2) → (3,3) → (4,4)</code>。</p> <br><p><strong>示例 4：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">grid = [[1]]</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">1</span></p> <br><p><strong>解释：</strong></p> <br><p>最长的 V 形对角线段长度为 1，路径如下：<code>(0,0)</code>。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>n == grid.length</code></li><li><code>m == grid[i].length</code></li><li><code>1 <= n, m <= 500</code></li><li><code>grid[i][j]</code> 的值为 <code>0</code>、<code>1</code> 或 <code>2</code>。</li></ul> <br><h3><a id="_94"></a>解题方法：记忆化搜索</h3> <br><h4><a id="_96"></a>解题思路</h4> <br><p>题目翻译：从一个位置出发沿4个45度倾斜的方向移动，最多<strong>右转</strong>一次，起点一定是<code>1</code>，经过路径一定是<code>2020...</code>(不能是<code>0202..</code>)，求最长路径是多少。</p> <br><p>写一个函数<code>dfs(int i, int j, int d, int times)</code>，代表当前位置坐标为<code>(i, j)</code>、前进方向为<code>d</code>、拐弯了<code>times</code>次的状态下，算上当前位置的最长路径长度。</p> <br><p>如果沿当前方向向前走能继续走，那就尝试；如果右转90度能继续走，那就也试试。终止条件：一定会走到走不动为止。</p> <br><p>那么，所有<code>grid[i][j] == 1</code>的下标<code>(i, j)</code>，4个方向且<code>times</code>为<code>0</code>的初始状态的到的最大结果即为答案。</p> <br><p>然后记得记忆化一下，算过的<code>(i, j, d, times)</code>不要再重复计算一次。</p> <br><h4><a id="_108"></a>具体方法</h4> <br><p>实际编码的时候，给定的数据量在使用默认的缓存时可能会TLE或OOM。所以C++和Python要使用<strong>数组</strong>手动记忆化一下。</p> <br><p>如何将<code>(i, j, d, times)</code>映射为一个下标？<code>i * m * 8 + j * 8 + d * 2 + times</code>即可。</p> <br><h4><a id="_114"></a>剪枝(可选)</h4> <br><p>有没有可以提前退出的时刻？有，可以维护一个“全局”的最长路径，若当前状态一直往前走无论如何也无法超过历史最优解，索性放弃尝试，<b title="可能不只是对题解，也适用于我">及时止损</b></p> <br><h4><a
