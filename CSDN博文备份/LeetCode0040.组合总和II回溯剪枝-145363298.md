<h2><a id="LetMeFly40_II___1"></a>【LetMeFly】40.组合总和 II：回溯 + 剪枝</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/combination-sum-ii/" rel="nofollow">https://leetcode.cn/problems/combination-sum-ii/</a></p> <br><p>给定一个候选人编号的集合 <code>candidates</code> 和一个目标数 <code>target</code> ，找出 <code>candidates</code> 中所有可以使数字和为 <code>target</code> 的组合。</p> <br><p><code>candidates</code> 中的每个数字在每个组合中只能使用 <strong>一次</strong> 。</p> <br><p><strong>注意：</strong>解集不能包含重复的组合。 </p> <br><p> </p> <br><p><strong>示例 1:</strong></p> <br><pre><strong>输入:</strong> candidates = <code>[10,1,2,7,6,1,5]</code>, target = <code>8</code>,<br><strong>输出:</strong><br>[<br>[1,1,6],<br>[1,2,5],<br>[1,7],<br>[2,6]<br>]</pre> <br><p><strong>示例 2:</strong></p> <br><pre><strong>输入:</strong> candidates = [2,5,2,1,2], target = 5,<br><strong>输出:</strong><br>[<br>[1,2,2],<br>[5]<br>]</pre> <br><p> </p> <br><p><strong>提示:</strong></p> <br><ul><li><code>1 <= candidates.length <= 100</code></li><li><code>1 <= candidates[i] <= 50</code></li><li><code>1 <= target <= 30</code></li></ul> <br><h3><a id="_47"></a>解题方法：回溯（剪枝）</h3> <br><p>类似<a href="https://blog.letmefly.xyz/2024/04/20/LeetCode%200039.%E7%BB%84%E5%90%88%E6%80%BB%E5%92%8C/" rel="nofollow">39.组合总和：回溯 + 剪枝</a>，但这道题比较困难的地方在于，<code>candidates</code>中有重复的元素，而答案中不能有重复的数组。</p> <br><p>怎么办呢，排序呗。刚开始还和之前一样走正常流程：</p> <br><ol><li>如果target已经达到则将当前方案加入答案数组。</li><li>如果已超target则直接返回</li><li>选当前元素并回溯</li><li>不选当前元素</li></ol> <br><p>不同之处在于，不选当前元素时，要保证选择的下一个元素和当前元素不同。</p> <br><blockquote> <br> <p>例如<code>[4, 4, 5]</code>，不选第一个<code>4</code>的话，就不能选第二个<code>4</code>。</p> <br> <p>否则的话，可能会出现<code>第一个4和5</code>、<code>第二个4和5</code>这两种相同的方案。</p> <br></blockquote>
