<h2><a id="LetMeFly1975_1"></a>【LetMeFly】1975.最大方阵和：脑筋急转弯</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/maximum-matrix-sum/" rel="nofollow">https://leetcode.cn/problems/maximum-matrix-sum/</a></p> <br><p>给你一个 <code>n x n</code> 的整数方阵 <code>matrix</code> 。你可以执行以下操作 <strong>任意次</strong> ：</p> <br><ul><li>选择 <code>matrix</code> 中 <strong>相邻</strong> 两个元素，并将它们都 <strong>乘以</strong> <code>-1</code> 。</li></ul> <br><p>如果两个元素有 <strong>公共边</strong> ，那么它们就是 <strong>相邻</strong> 的。</p> <br><p>你的目的是 <strong>最大化</strong> 方阵元素的和。请你在执行以上操作之后，返回方阵的 <strong>最大</strong> 和。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><img alt="" src="https://i-blog.csdnimg.cn/img_convert/dd304ee6f2054a6f1c6d72d7fa7682db.png" /> <br><pre><b>输入：</b>matrix = [[1,-1],[-1,1]]<br><b>输出：</b>4<br><b>解释：</b>我们可以执行以下操作使和等于 4 ：<br>- 将第一行的 2 个元素乘以 -1 。<br>- 将第一列的 2 个元素乘以 -1 。<br></pre> <br><p><strong>示例 2：</strong></p> <br><img alt="" src="https://i-blog.csdnimg.cn/img_convert/4ea0294e5b3a66add794e5b558a3b66c.png" /> <br><pre><b>输入：</b>matrix = [[1,2,3],[-1,-2,-3],[1,2,3]]<br><b>输出：</b>16<br><b>解释：</b>我们可以执行以下操作使和等于 16 ：<br>- 将第二行的最后 2 个元素乘以 -1 。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>n == matrix.length == matrix[i].length</code></li><li><code>2 <= n <= 250</code></li><li><code>-10<sup>5</sup> <= matrix[i][j] <= 10<sup>5</sup></code></li></ul> <br><h3><a id="_46"></a>解题方法：脑筋急转弯</h3> <br><p>解这道题只需要明白：</p> <br><ol><li>只能相邻取负 等价于 可任意两元素取负</li></ol> <br><p>就够了。</p> <br><p>But Why?</p> <br><p>知道链式效应么（名字瞎起的），<code>[a, b, c, d]</code>中若想实现<code>a, d</code>取负则可通过<code>a, b</code>、<code>b, c</code>、<code>c, d</code>取负的方式实现，其中中间的<code>b</code>、<code>c</code>都取负了偶数次，相当于没变。</p> <br><p>所以我们可以得到：</p> <br><ul><li>若matrix中有偶数个负数，那么两两配对并取反，相当于全是正数！</li><li>若matrix中有奇数个负数，那么最终一定会有一个负数无法通过负数们“内部消化”，这个负数找一个矩阵中绝对值最小的数并让它当负数自己当正数就好了。</li></ul> <br><p>具体而言，遍历一遍矩阵，累加每个数的绝对值，并记下绝对值最小的数、负数的个数。</p>
