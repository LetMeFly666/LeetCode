<h2><a id="LetMeFly2209DFS_1"></a>【LetMeFly】2209.用地毯覆盖后的最少白色砖块：记忆化搜索之——深度优先搜索(DFS)</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/minimum-white-tiles-after-covering-with-carpets/" rel="nofollow">https://leetcode.cn/problems/minimum-white-tiles-after-covering-with-carpets/</a></p> <br><p>给你一个下标从<strong> 0</strong> 开始的 <strong>二进制</strong> 字符串 <code>floor</code> ，它表示地板上砖块的颜色。</p> <br><ul><li><code>floor[i] = '0'</code> 表示地板上第 <code>i</code> 块砖块的颜色是 <strong>黑色</strong> 。</li><li><code>floor[i] = '1'</code> 表示地板上第 <code>i</code> 块砖块的颜色是 <strong>白色</strong> 。</li></ul> <br><p>同时给你 <code>numCarpets</code> 和 <code>carpetLen</code> 。你有 <code>numCarpets</code> 条 <strong>黑色</strong> 的地毯，每一条 <strong>黑色</strong> 的地毯长度都为 <code>carpetLen</code> 块砖块。请你使用这些地毯去覆盖砖块，使得未被覆盖的剩余 <strong>白色</strong> 砖块的数目 <strong>最小</strong> 。地毯相互之间可以覆盖。</p> <br><p>请你返回没被覆盖的白色砖块的 <strong>最少</strong> 数目。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/a37188bac190b3f13db176227147101d.png" /></p> <br><pre><b>输入：</b>floor = "10110101", numCarpets = 2, carpetLen = 2<br><b>输出：</b>2<br><b>解释：</b><br>上图展示了剩余 2 块白色砖块的方案。<br>没有其他方案可以使未被覆盖的白色砖块少于 2 块。<br></pre> <br><p><strong>示例 2：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/b6934e0d72121251a035a001cd528147.png" /></p> <br><pre><b>输入：</b>floor = "11111", numCarpets = 2, carpetLen = 3<br><b>输出：</b>0<br><b>解释：</b><br>上图展示了所有白色砖块都被覆盖的一种方案。<br>注意，地毯相互之间可以覆盖。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= carpetLen <= floor.length <= 1000</code></li><li><code>floor[i]</code> 要么是 <code>'0'</code> ，要么是 <code>'1'</code> 。</li><li><code>1 <= numCarpets <= 1000</code></li></ul> <br><h3><a id="DFS_52"></a>解题方法：深度优先搜索(DFS)</h3> <br><p>写一个函数<code>dfs(n, index)</code>，代表使用<code>n</code>块地毯从下标<code>index</code>开始往后覆盖剩余的最小白色砖块数。</p> <br><ul><li> <p>如果不覆盖当前砖块：</p> <p><code>dfs(n, index + 1) + (floor[index] == '1')</code></p> <br>  <ol><li>可以使用<code>n</code>块地毯覆盖后续砖块，最少有<code>dfs(n, index + 1)</code>块白砖不能覆盖</li><li>如果这块砖是白色，则未覆盖白砖数量加一</li></ol> </li><li> <p>如果覆盖当前砖块：</p> <p><code>dfs(n - 1, index + 地毯长度)</code></p> <br>  <ol><li>可以使用<code>n - 1</code>块地毯覆盖从<code>index + 地毯长度</code>开始的砖块</li></ol> </li></ul> <br><p>终止条件：当前地毯能覆盖剩下所有砖块。</p> <br><p>然后，再使用一个缓存记忆化一下就好了。</p>
