<h2><a id="LetMeFly3567_1"></a>【LetMeFly】3567.子矩阵的最小绝对差：暴力模拟</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/minimum-absolute-difference-in-sliding-submatrix/" rel="nofollow">https://leetcode.cn/problems/minimum-absolute-difference-in-sliding-submatrix/</a></p> <br><p>给你一个 <code>m x n</code> 的整数矩阵 <code>grid</code> 和一个整数 <code>k</code>。</p> <br><p>对于矩阵 <code>grid</code> 中的每个连续的 <code>k x k</code> <strong>子矩阵</strong>，计算其中任意两个 <strong>不同</strong>值 之间的 <strong>最小绝对差 </strong>。</p> <br><p>返回一个大小为 <code>(m - k + 1) x (n - k + 1)</code> 的二维数组 <code>ans</code>，其中 <code>ans[i][j]</code> 表示以 <code>grid</code> 中坐标 <code>(i, j)</code> 为左上角的子矩阵的最小绝对差。</p> <br><p><strong>注意</strong>：如果子矩阵中的所有元素都相同，则答案为 0。</p> <br><p>子矩阵 <code>(x1, y1, x2, y2)</code> 是一个由选择矩阵中所有满足 <code>x1 <= x <= x2</code> 且 <code>y1 <= y <= y2</code> 的单元格 <code>matrix[x][y]</code> 组成的矩阵。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">grid = [[1,8],[3,-2]], k = 2</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">[[2]]</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>只有一个可能的 <code>k x k</code> 子矩阵：<code><span class="example-io">[[1, 8], [3, -2]]</span></code>。</li><li>子矩阵中的不同值为 <code>[1, 8, 3, -2]</code>。</li><li>子矩阵中的最小绝对差为 <code>|1 - 3| = 2</code>。因此，答案为 <code>[[2]]</code>。</li></ul> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">grid = [[3,-1]], k = 1</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">[[0,0]]</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>每个 <code>k x k</code> 子矩阵中只有一个不同的元素。</li><li>因此，答案为 <code>[[0, 0]]</code>。</li></ul> <br><p><strong class="example">示例 3：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">grid = [[1,-2,3],[2,3,5]], k = 2</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">[[1,2]]</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>有两个可能的 <code>k × k</code> 子矩阵： <pre><code><ul><br>	<li>以 <code>(0, 0)</code> 为起点的子矩阵：<code>[[1, -2], [2, 3]]</code>。<br><br>	<ul><br>		<li>子矩阵中的不同值为 <code>[1, -2, 2, 3]</code>。</li><br>		<li>子矩阵中的最小绝对差为 <code>|1 - 2| = 1</code>。</li><br>	</ul><br>	</li><br>	<li>以 <code>(0, 1)</code> 为起点的子矩阵：<code>[[-2, 3], [3, 5]]</code>。<br>	<ul><br>		<li>子矩阵中的不同值为 <code>[-2, 3, 5]</code>。</li><br>		<li>子矩阵中的最小绝对差为 <code>|3 - 5| = 2</code>。</li><br>	</ul><br>	</li><br></ul><br></li><br><li>因此，答案为 <code>[[1, 2]]</code>。</li><br></code></pre> </li></ul> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= m == grid.length <= 30</code></li><li><code>1 <= n == grid[i].length <= 30</code></li><li><code>-10<sup>5</sup> <= grid[i][j] <= 10<sup>5</sup></code></li><li><code>1 <= k <= min(m, n)</code></li></ul> <br><h3><a id="_93"></a>解题方法：暴力模拟</h3> <br><p>大小为<span
