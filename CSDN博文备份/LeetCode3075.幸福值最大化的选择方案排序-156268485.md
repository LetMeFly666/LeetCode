<h2><a id="LetMeFly3075_1"></a>【LetMeFly】3075.幸福值最大化的选择方案：排序</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/maximize-happiness-of-selected-children/" rel="nofollow">https://leetcode.cn/problems/maximize-happiness-of-selected-children/</a></p> <br><p>给你一个长度为 <code>n</code> 的数组 <code>happiness</code> ，以及一个<strong> 正整数 </strong><code>k</code> 。</p> <br><p><code>n</code> 个孩子站成一队，其中第 <code>i</code> 个孩子的 <strong>幸福值</strong> 是<strong> </strong><code>happiness[i]</code> 。你计划组织 <code>k</code> 轮筛选从这 <code>n</code> 个孩子中选出 <code>k</code> 个孩子。</p> <br><p>在每一轮选择一个孩子时，所有<strong> 尚未 </strong>被选中的孩子的 <strong>幸福值 </strong>将减少 <code>1</code> 。注意，幸福值<strong> 不能 </strong>变成负数，且只有在它是正数的情况下才会减少。</p> <br><p>选择 <code>k</code> 个孩子，并使你选中的孩子幸福值之和最大，返回你能够得到的<strong> </strong><strong>最大值</strong> 。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><pre><strong>输入：</strong>happiness = [1,2,3], k = 2<br><strong>输出：</strong>4<br><strong>解释：</strong>按以下方式选择 2 个孩子：<br>- 选择幸福值为 3 的孩子。剩余孩子的幸福值变为 [0,1] 。<br>- 选择幸福值为 1 的孩子。剩余孩子的幸福值变为 [0] 。注意幸福值不能小于 0 。<br>所选孩子的幸福值之和为 3 + 1 = 4 。<br></pre> <br><p><strong class="example">示例 2：</strong></p> <br><pre><strong>输入：</strong>happiness = [1,1,1,1], k = 2<br><strong>输出：</strong>1<br><strong>解释：</strong>按以下方式选择 2 个孩子：<br>- 选择幸福值为 1 的任意一个孩子。剩余孩子的幸福值变为 [0,0,0] 。<br>- 选择幸福值为 0 的孩子。剩余孩子的幸福值变为 [0,0] 。<br>所选孩子的幸福值之和为 1 + 0 = 1 。<br></pre> <br><p><strong class="example">示例 3：</strong></p> <br><pre><strong>输入：</strong>happiness = [2,3,4,5], k = 1<br><strong>输出：</strong>5<br><strong>解释：</strong>按以下方式选择 1 个孩子：<br>- 选择幸福值为 5 的孩子。剩余孩子的幸福值变为 [1,2,3] 。<br>所选孩子的幸福值之和为 5 。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= n == happiness.length <= 2 * 10<sup>5</sup></code></li><li><code>1 <= happiness[i] <= 10<sup>8</sup></code></li><li><code>1 <= k <= n</code></li></ul> <br><h3><a
