rel="nofollow">https://leetcode.cn/problems/mirror-distance-of-an-integer/</a></p> <br><p>给你一个整数 <code>n</code>。</p> <br><p>定义它的 <strong>镜像距离</strong> 为：<code>abs(n - reverse(n))</code>​​​​​​​，其中 <code>reverse(n)</code> 表示将 <code>n</code> 的数字反转后形成的整数。</p> <br><p>返回表示 <code>n</code> 的镜像距离的整数。</p> <br><p>其中，<code>abs(x)</code> 表示 <code>x</code> 的绝对值。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">n = 25</span></p> <br></div><br><p><strong>输出：</strong> <span 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">n = 10</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">9</span></p> <br><p><strong>解释：</strong></p> <br><ul><li><code>reverse(7) = 7</code>。</li><li>因此，答案为 <code>abs(7 - 7) = 0</code>。</li></ul> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= n <=
