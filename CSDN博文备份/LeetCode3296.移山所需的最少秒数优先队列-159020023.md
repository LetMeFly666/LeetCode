<h2><a id="LetMeFly3296_1"></a>【LetMeFly】3296.移山所需的最少秒数：优先队列</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/minimum-number-of-seconds-to-make-mountain-height-zero/" rel="nofollow">https://leetcode.cn/problems/minimum-number-of-seconds-to-make-mountain-height-zero/</a></p> <br><p>给你一个整数 <code>mountainHeight</code> 表示山的高度。</p> <br><p>同时给你一个整数数组 <code>workerTimes</code>，表示工人们的工作时间（单位：<strong>秒</strong>）。</p> <br><p>工人们需要 <strong>同时 </strong>进行工作以 <strong>降低 </strong>山的高度。对于工人 <code>i</code> :</p> <br><ul><li>山的高度降低 <code>x</code>，需要花费 <code>workerTimes[i] + workerTimes[i] * 2 + ... + workerTimes[i] * x</code> 秒。例如： <pre><code><ul><br>	<li>山的高度降低 1，需要 <code>workerTimes[i]</code> 秒。</li><br>	<li>山的高度降低 2，需要 <code>workerTimes[i] + workerTimes[i] * 2</code> 秒，依此类推。</li><br></ul><br></li><br></code></pre> </li></ul> <br><p>返回一个整数，表示工人们使山的高度降低到 0 所需的 <strong>最少</strong> 秒数。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">mountainHeight = 4, workerTimes = [2,1,1]</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">3</span></p> <br><p><strong>解释：</strong></p> <br><p>将山的高度降低到 0 的一种方式是：</p> <br><ul><li>工人 0 将高度降低 1，花费 <code>workerTimes[0] = 2</code> 秒。</li><li>工人 1 将高度降低 2，花费 <code>workerTimes[1] + workerTimes[1] * 2 = 3</code> 秒。</li><li>工人 2 将高度降低 1，花费 <code>workerTimes[2] = 1</code> 秒。</li></ul> <br><p>因为工人同时工作，所需的最少时间为 <code>max(2, 3, 1) = 3</code> 秒。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">mountainHeight = 10, workerTimes = [3,2,2,4]</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">12</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>工人 0 将高度降低 2，花费 <code>workerTimes[0] + workerTimes[0] * 2 = 9</code> 秒。</li><li>工人 1 将高度降低 3，花费 <code>workerTimes[1] + workerTimes[1] * 2 + workerTimes[1] * 3 = 12</code> 秒。</li><li>工人 2 将高度降低 3，花费 <code>workerTimes[2] + workerTimes[2] * 2 + workerTimes[2] * 3 = 12</code> 秒。</li><li>工人 3 将高度降低 2，花费 <code>workerTimes[3] + workerTimes[3] * 2 = 12</code> 秒。</li></ul> <br><p>所需的最少时间为 <code>max(9, 12, 12, 12) = 12</code> 秒。</p> <br><p><strong class="example">示例 3：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">mountainHeight = 5, workerTimes = [1]</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">15</span></p> <br><p><strong>解释：</strong></p> <br><p>这个示例中只有一个工人，所以答案是 <code>workerTimes[0] + workerTimes[0] * 2 + workerTimes[0] * 3 + workerTimes[0] * 4 + workerTimes[0] * 5 = 15</code> 秒。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= mountainHeight <= 10<sup>5</sup></code></li><li><code>1 <= workerTimes.length <= 10<sup>4</sup></code></li><li><code>1 <= workerTimes[i] <= 10<sup>6</sup></code></li></ul> <br><h3><a id="_88"></a>解题方法：优先队列</h3> <br><p>使用一个优先队列，每次选完工最早的工人降低1个山高。工人完工后可以再次进入队列，只不过再次进入队列后的下次工作耗时更久而已。</p> <br><p>具体来说，优先队列中存放每个工人的<code>(完工时间, 共计降低高度, baseTime)</code>，<code>共计降低高度</code>和<code>baseTime</code>都是为了计算完工时间。</p>
