<h2><a id="LetMeFly3074_1"></a>【LetMeFly】3074.重新分装苹果：排序</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/apple-redistribution-into-boxes/" rel="nofollow">https://leetcode.cn/problems/apple-redistribution-into-boxes/</a></p> <br><p>给你一个长度为 <code>n</code> 的数组 <code>apple</code> 和另一个长度为 <code>m</code> 的数组 <code>capacity</code> 。</p> <br><p>一共有 <code>n</code> 个包裹，其中第 <code>i</code> 个包裹中装着 <code>apple[i]</code> 个苹果。同时，还有 <code>m</code> 个箱子，第 <code>i</code> 个箱子的容量为 <code>capacity[i]</code> 个苹果。</p> <br><p>请你选择一些箱子来将这 <code>n</code> 个包裹中的苹果重新分装到箱子中，返回你需要选择的箱子的<strong> 最小</strong> 数量。</p> <br><p><strong>注意</strong>，同一个包裹中的苹果可以分装到不同的箱子中。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><pre><strong>输入：</strong>apple = [1,3,2], capacity = [4,3,1,5,2]<br><strong>输出：</strong>2<br><strong>解释：</strong>使用容量为 4 和 5 的箱子。<br>总容量大于或等于苹果的总数，所以可以完成重新分装。<br></pre> <br><p><strong class="example">示例 2：</strong></p> <br><pre><strong>输入：</strong>apple = [5,5,5], capacity = [2,4,2,7]<br><strong>输出：</strong>4<br><strong>解释：</strong>需要使用所有箱子。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= n == apple.length <= 50</code></li><li><code>1 <= m == capacity.length <= 50</code></li><li><code>1 <= apple[i], capacity[i] <= 50</code></li><li>输入数据保证可以将包裹中的苹果重新分装到箱子中。</li></ul> <br><h3><a id="_45"></a>解题方法：排序</h3> <br><p>每个<font title="2025.12.24">苹果</font>都要有它的归宿，一共有多少个苹果呢？算一算就知道了。</p> <br><p>如何使用最少的箱子？当然是用尽可能大的箱子！</p>
