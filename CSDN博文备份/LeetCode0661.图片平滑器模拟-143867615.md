<h2><a id="LetMeFly661_1"></a>【LetMeFly】661.图片平滑器：模拟</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/image-smoother/" rel="nofollow">https://leetcode.cn/problems/image-smoother/</a></p> <br><p><strong>图像平滑器</strong> 是大小为 <code>3 x 3</code> 的过滤器，用于对图像的每个单元格平滑处理，平滑处理后单元格的值为该单元格的平均灰度。</p> <br><p>每个单元格的<strong>  平均灰度</strong> 定义为：该单元格自身及其周围的 8 个单元格的平均值，结果需向下取整。（即，需要计算蓝色平滑器中 9 个单元格的平均值）。</p> <br><p>如果一个单元格周围存在单元格缺失的情况，则计算平均灰度时不考虑缺失的单元格（即，需要计算红色平滑器中 4 个单元格的平均值）。</p> <br><p><img src="https://i-blog.csdnimg.cn/direct/c0079440db3d44c182a1a7505fa91db1.png" alt="在这里插入图片描述" /></p> <br><p>给你一个表示图像灰度的 <code>m x n</code> 整数矩阵 <code>img</code> ，返回对图像的每个单元格平滑处理后的图像 。</p> <br><p> </p> <br><p><strong>示例 1:</strong></p> <br><p><img src="https://i-blog.csdnimg.cn/direct/0535b69bc6c343f3a2e3caa5c747939c.png" alt="在这里插入图片描述" /></p> <br><pre><strong>输入:</strong>img = [[1,1,1],[1,0,1],[1,1,1]]<br><strong>输出:</strong>[[0, 0, 0],[0, 0, 0], [0, 0, 0]]<br><strong>解释:</strong><br>对于点 (0,0), (0,2), (2,0), (2,2): 平均(3/4) = 平均(0.75) = 0<br>对于点 (0,1), (1,0), (1,2), (2,1): 平均(5/6) = 平均(0.83333333) = 0<br>对于点 (1,1): 平均(8/9) = 平均(0.88888889) = 0<br></pre> <br><p><strong>示例 2:</strong></p> <br><p><img src="https://i-blog.csdnimg.cn/direct/48d226238db34b39bbf7712fe4791185.png" alt="在这里插入图片描述" /></p> <br><pre><strong>输入:</strong> img = [[100,200,100],[200,50,200],[100,200,100]]<br><strong>输出:</strong> [[137,141,137],[141,138,141],[137,141,137]]<br><strong>解释:</strong><br>对于点 (0,0), (0,2), (2,0), (2,2): floor((100+200+200+50)/4) = floor(137.5) = 137<br>对于点 (0,1), (1,0), (1,2), (2,1): floor((200+200+50+200+100+100)/6) = floor(141.666667) = 141<br>对于点 (1,1): floor((50+200+200+200+200+100+100+100+100)/9) = floor(138.888889) = 138<br></pre> <br><p> </p> <br><p><strong>提示:</strong></p> <br><ul><li><code>m == img.length</code></li><li><code>n == img[i].length</code></li><li><code>1 <= m, n <= 200</code></li><li><code>0 <=
