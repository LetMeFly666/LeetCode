href="https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-using-strategy/" rel="nofollow">https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-using-strategy/</a></p> <br><p>给你两个整数数组 <code>prices</code> 和 <code>strategy</code>，其中：</p> <br><ul><li><code>prices[i]</code> 表示第 <code>i</code> 天某股票的价格。</li><li><code>strategy[i]</code> 表示第 <code>i</code> 天的交易策略，其中： <br>  <ul><li><code>-1</code> 表示买入一单位股票。</li><li><code>0</code> 表示持有股票。</li><li><code>1</code> 表示卖出一单位股票。</li></ul> </li></ul> <br><p>同时给你一个 <strong>偶数 </strong>整数 <code>k</code>，你可以对 <code>strategy</code> 进行 <strong>最多一次 </strong>修改。一次修改包括：</p> <br><ul><li>选择 <code>strategy</code> 中恰好 <code>k</code> 个 <strong>连续 </strong>元素。</li><li>将前 <code>k / 2</code> 个元素设为 <code>0</code>（持有）。</li><li>将后 <code>k / 2</code> 个元素设为 <code>1</code>（卖出）。</li></ul> <br><p><strong>利润 </strong>定义为所有天数中 <code>strategy[i] * prices[i]</code> 的 <strong>总和 </strong>。</p> <br><p>返回你可以获得的 <strong>最大 </strong>可能利润。</p> <br><p><strong>注意：</strong> 没有预算或股票持有数量的限制，因此所有买入和卖出操作均可行，无需考虑过去的操作。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">prices = [4,2,8], strategy = [-1,0,1], k = 2</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">10</span></p> <br><p><strong>解释：</strong></p> <br><table><thead><tr><th>修改</th><th>策略</th><th>利润计算</th><th>利润</th></tr></thead><tbody><tr><td>原始</td><td>[-1, 0, 1]</td><td>(-1 × 4) + (0 × 2) + (1 × 8) = -4 + 0 + 8</td><td>4</td></tr><tr><td>修改 [0, 1]</td><td>[0, 1, 1]</td><td>(0 × 4) + (1 × 2) + (1 × 8) = 0 + 2 + 8</td><td>10</td></tr><tr><td>修改 [1, 2]</td><td>[-1, 0, 1]</td><td>(-1 × 4) + (0 × 2) + (1 × 8) = -4 + 0 + 8</td><td>4</td></tr></tbody></table> <br><p>因此，最大可能利润是 10，通过修改子数组 <code>[0, 1]</code> 实现。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">prices = [5,4,3], strategy = [1,1,0], k = 2</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">9</span></p> <br><p><strong>解释：</strong></p> <br><div class="example-block"> <br> <table><thead><tr><th>修改</th><th>策略</th><th>利润计算</th><th>利润</th></tr></thead><tbody><tr><td>原始</td><td>[1, 1, 0]</td><td>(1 × 5) + (1 × 4) + (0 × 3) = 5 + 4 + 0</td><td>9</td></tr><tr><td>修改 [0, 1]</td><td>[0, 1, 0]</td><td>(0 × 5) + (1 × 4) + (0 × 3) = 0 + 4 + 0</td><td>4</td></tr><tr><td>修改 [1, 2]</td><td>[1, 0, 1]</td><td>(1 × 5) + (0 × 4) + (1 × 3) = 5 + 0 + 3</td><td>8</td></tr></tbody></table> <br></div><br><p>因此，最大可能利润是 9，无需任何修改即可达成。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>2 <= prices.length == strategy.length <= 10<sup>5</sup></code></li><li><code>1 <= prices[i] <= 10<sup>5</sup></code></li><li><code>-1 <= strategy[i] <= 1</code></li><li><code>2 <= k <= prices.length</code></li><li><code>k</code> 是偶数</li></ul> <br><h3><a id="_143"></a>解题方法：滑动窗口</h3> <br><p>既然修改范围是定长的，并且最多修改1次，那么就从前往后将每一种修改可能都试试呗。</p>
