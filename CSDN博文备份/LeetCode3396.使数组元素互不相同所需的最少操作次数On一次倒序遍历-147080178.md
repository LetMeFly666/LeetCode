<h2><a id="LetMeFly3396On_1"></a>【LetMeFly】3396.使数组元素互不相同所需的最少操作次数：O(n)一次倒序遍历</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/minimum-number-of-operations-to-make-elements-in-array-distinct/" rel="nofollow">https://leetcode.cn/problems/minimum-number-of-operations-to-make-elements-in-array-distinct/</a></p> <br><p>给你一个整数数组 <code>nums</code>，你需要确保数组中的元素 <strong>互不相同 </strong>。为此，你可以执行以下操作任意次：</p> <br><ul><li>从数组的开头移除 3 个元素。如果数组中元素少于 3 个，则移除所有剩余元素。</li></ul> <br><p><strong>注意：</strong>空数组也视作为数组元素互不相同。返回使数组元素互不相同所需的 <strong>最少操作次数 </strong>。 <br> </p> <br><p> </p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">nums = [1,2,3,4,2,3,3,5,7]</span></p> <br></div> <br><p><strong>输出：</strong> <span class="example-io">2</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>第一次操作：移除前 3 个元素，数组变为 <code>[4, 2, 3, 3, 5, 7]</code>。</li><li>第二次操作：再次移除前 3 个元素，数组变为 <code>[3, 5, 7]</code>，此时数组中的元素互不相同。</li></ul> <br><p>因此，答案是 2。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">nums = [4,5,6,4,4]</span></p> <br></div> <br><p><strong>输出：</strong> <span class="example-io">2</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>第一次操作：移除前 3 个元素，数组变为 <code>[4, 4]</code>。</li><li>第二次操作：移除所有剩余元素，数组变为空。</li></ul> <br><p>因此，答案是 2。</p> <br><p><strong class="example">示例 3：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">nums = [6,7,8,9]</span></p> <br></div> <br><p><strong>输出：</strong> <span class="example-io">0</span></p> <br><p><strong>解释：</strong></p> <br><p>数组中的元素已经互不相同，因此不需要进行任何操作，答案是 0。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= nums.length <= 100</code></li><li><code>1 <= nums[i] <= 100</code></li></ul> <br><h3><a id="_74"></a>解题方法：遍历</h3> <br><p>只有一种删除重复元素的方式，就是把开头几个元素都删了。</p>
