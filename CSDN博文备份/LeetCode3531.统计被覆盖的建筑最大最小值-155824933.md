<h2><a id="LetMeFly3531_1"></a>【LetMeFly】3531.统计被覆盖的建筑：最大最小值</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/count-covered-buildings/" rel="nofollow">https://leetcode.cn/problems/count-covered-buildings/</a></p> <br><p>给你一个正整数 <code>n</code>，表示一个 <code>n x n</code> 的城市，同时给定一个二维数组 <code>buildings</code>，其中 <code>buildings[i] = [x, y]</code> 表示位于坐标 <code>[x, y]</code> 的一个 <strong>唯一 </strong>建筑。</p> <br><p>如果一个建筑在四个方向（左、右、上、下）中每个方向上都至少存在一个建筑，则称该建筑 <strong>被覆盖 </strong>。</p> <br><p>返回 <strong>被覆盖 </strong>的建筑数量。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><p><img src="https://i-blog.csdnimg.cn/img_convert/375ffe8ac9cdcda38c65bfb8f826699c.jpeg" /></p> <br><div class="example-block"> <br> <p><strong>输入:</strong> <span class="example-io">n = 3, buildings = [[1,2],[2,2],[3,2],[2,1],[2,3]]</span></p> <br></div><br><p><strong>输出:</strong> <span class="example-io">1</span></p> <br><p><strong>解释:</strong></p> <br><ul><li>只有建筑 <code>[2,2]</code> 被覆盖，因为它在每个方向上都至少存在一个建筑： <pre><code><ul><br>	<li>上方 (<code>[1,2]</code>)</li><br>	<li>下方 (<code>[3,2]</code>)</li><br>	<li>左方 (<code>[2,1]</code>)</li><br>	<li>右方 (<code>[2,3]</code>)</li><br></ul><br></li><br><li>因此，被覆盖的建筑数量是 1。</li><br></code></pre> </li></ul> <br><p><strong class="example">示例 2：</strong></p> <br><p><img src="https://i-blog.csdnimg.cn/img_convert/e4f8f96bb4597ecd014bf5c14310095e.jpeg" /></p> <br><div class="example-block"> <br> <p><strong>输入:</strong> <span class="example-io">n = 3, buildings = [[1,1],[1,2],[2,1],[2,2]]</span></p> <br></div><br><p><strong>输出:</strong> <span class="example-io">0</span></p> <br><p><strong>解释:</strong></p> <br><ul><li>没有任何一个建筑在每个方向上都有至少一个建筑。</li></ul> <br><p><strong class="example">示例 3：</strong></p> <br><p><img src="https://i-blog.csdnimg.cn/img_convert/0c5731328b73b0ed99acdc7f46b22f74.jpeg" /></p> <br><div class="example-block"> <br> <p><strong>输入:</strong> <span class="example-io">n = 5, buildings = [[1,3],[3,2],[3,3],[3,5],[5,3]]</span></p> <br></div><br><p><strong>输出:</strong> <span class="example-io">1</span></p> <br><p><strong>解释:</strong></p> <br><ul><li>只有建筑 <code>[3,3]</code> 被覆盖，因为它在每个方向上至少存在一个建筑： <pre><code><ul><br>	<li>上方 (<code>[1,3]</code>)</li><br>	<li>下方 (<code>[5,3]</code>)</li><br>	<li>左方 (<code>[3,2]</code>)</li><br>	<li>右方 (<code>[3,5]</code>)</li><br></ul><br></li><br><li>因此，被覆盖的建筑数量是 1。</li><br></code></pre> </li></ul> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>2 <= n <= 10<sup>5</sup></code></li><li><code>1 <= buildings.length <= 10<sup>5</sup></code></li><li><code>buildings[i] = [x, y]</code></li><li><code>1 <= x, y <=
