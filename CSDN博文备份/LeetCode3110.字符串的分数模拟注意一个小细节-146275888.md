<h2><a id="LetMeFly3110_1"></a>【LetMeFly】3110.字符串的分数：模拟(注意一个小细节)</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/score-of-a-string/" rel="nofollow">https://leetcode.cn/problems/score-of-a-string/</a></p> <br><p>给你一个字符串 <code>s</code> 。一个字符串的 <strong>分数</strong> 定义为相邻字符 <strong>ASCII</strong> 码差值绝对值的和。</p> <br><p>请你返回 <code>s</code> 的 <strong>分数</strong> 。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>s = "hello"</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>13</span></p> <br><p><strong>解释：</strong></p> <br><p><code>s</code> 中字符的 <strong>ASCII </strong>码分别为：<code>'h' = 104</code> ，<code>'e' = 101</code> ，<code>'l' = 108</code> ，<code>'o' = 111</code> 。所以 <code>s</code> 的分数为 <code>|104 - 101| + |101 - 108| + |108 - 108| + |108 - 111| = 3 + 7 + 0 + 3 = 13</code> 。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>s = "zaz"</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>50</span></p> <br><p><strong>解释：</strong></p> <br><p><code>s</code> 中字符的 <strong>ASCII </strong>码分别为：<code>'z' = 122</code> ，<code>'a' = 97</code> 。所以 <code>s</code> 的分数为 <code>|122 - 97| + |97 - 122| = 25 + 25 = 50</code> 。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>2 <= s.length <= 100</code></li><li><code>s</code> 只包含小写英文字母。</li></ul> <br><h3><a id="_46"></a>解题方法：模拟</h3> <br><p>直接从第二个字符开始遍历字符串，将这个字符减去前一个字符的绝对值累加到答案中，最终返回即可。</p> <br><p>注意：请谨慎使用8比特数减8比特数（如char - char或byte -
