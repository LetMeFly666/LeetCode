<h2><a id="LetMeFly2931_1"></a>【LetMeFly】2931.购买物品的最大开销：多指针（平均优于排序）</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/maximum-spending-after-buying-items/" rel="nofollow">https://leetcode.cn/problems/maximum-spending-after-buying-items/</a></p> <br><p>给你一个下标从 <strong>0</strong> 开始大小为 <code>m * n</code> 的整数矩阵 <code>values</code> ，表示 <code>m</code> 个不同商店里 <code>m * n</code> 件不同的物品。每个商店有 <code>n</code> 件物品，第 <code>i</code> 个商店的第 <code>j</code> 件物品的价值为 <code>values[i][j]</code> 。除此以外，第 <code>i</code> 个商店的物品已经按照价值非递增排好序了，也就是说对于所有 <code>0 <= j < n - 1</code> 都有 <code>values[i][j] >= values[i][j + 1]</code> 。</p> <br><p>每一天，你可以在一个商店里购买一件物品。具体来说，在第 <code>d</code> 天，你可以：</p> <br><ul><li>选择商店 <code>i</code> 。</li><li>购买数组中最右边的物品 <code>j</code> ，开销为 <code>values[i][j] * d</code> 。换句话说，选择该商店中还没购买过的物品中最大的下标 <code>j</code> ，并且花费 <code>values[i][j] * d</code> 去购买。</li></ul> <br><p><strong>注意</strong>，所有物品都视为不同的物品。比方说如果你已经从商店 <code>1</code> 购买了物品 <code>0</code> ，你还可以在别的商店里购买其他商店的物品 <code>0</code> 。</p> <br><p>请你返回购买所有 <code>m * n</code> 件物品需要的 <strong>最大开销</strong> 。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><pre><b>输入：</b>values = [[8,5,2],[6,4,1],[9,7,3]]<br><b>输出：</b>285<br><b>解释：</b>第一天，从商店 1 购买物品 2 ，开销为 values[1][2] * 1 = 1 。<br>第二天，从商店 0 购买物品 2 ，开销为 values[0][2] * 2 = 4 。<br>第三天，从商店 2 购买物品 2 ，开销为 values[2][2] * 3 = 9 。<br>第四天，从商店 1 购买物品 1 ，开销为 values[1][1] * 4 = 16 。<br>第五天，从商店 0 购买物品 1 ，开销为 values[0][1] * 5 = 25 。<br>第六天，从商店 1 购买物品 0 ，开销为 values[1][0] * 6 = 36 。<br>第七天，从商店 2 购买物品 1 ，开销为 values[2][1] * 7 = 49 。<br>第八天，从商店 0 购买物品 0 ，开销为 values[0][0] * 8 = 64 。<br>第九天，从商店 2 购买物品 0 ，开销为 values[2][0] * 9 = 81 。<br>所以总开销为 285 。<br>285 是购买所有 m * n 件物品的最大总开销。<br></pre> <br><p><strong class="example">示例 2：</strong></p> <br><pre><b>输入：</b>values = [[10,8,6,4,2],[9,7,5,3,2]]<br><b>输出：</b>386<br><b>解释：</b>第一天，从商店 0 购买物品 4 ，开销为 values[0][4] * 1 = 2 。<br>第二天，从商店 1 购买物品 4 ，开销为 values[1][4] * 2 = 4 。<br>第三天，从商店 1 购买物品 3 ，开销为 values[1][3] * 3 = 9 。<br>第四天，从商店 0 购买物品 3 ，开销为 values[0][3] * 4 = 16 。<br>第五天，从商店 1 购买物品 2 ，开销为 values[1][2] * 5 = 25 。<br>第六天，从商店 0 购买物品 2 ，开销为 values[0][2] * 6 = 36 。<br>第七天，从商店 1 购买物品 1 ，开销为 values[1][1] * 7 = 49 。<br>第八天，从商店 0 购买物品 1 ，开销为 values[0][1] * 8 = 64 。<br>第九天，从商店 1 购买物品 0 ，开销为 values[1][0] * 9 = 81 。<br>第十天，从商店 0 购买物品 0 ，开销为 values[0][0] * 10 = 100 。<br>所以总开销为 386 。<br>386 是购买所有 m * n 件物品的最大总开销。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= m == values.length <= 10</code></li><li><code>1 <= n == values[i].length <= 10<sup>4</sup></code></li><li><code>1 <= values[i][j] <= 10<sup>6</sup></code></li><li><code>values[i]</code> 按照非递增顺序排序。</li></ul> <br><h3><a
