<h2><a id="LetMeFly3238_1"></a>【LetMeFly】3238.求出胜利玩家的数目：哈希表计数</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/find-the-number-of-winning-players/" rel="nofollow">https://leetcode.cn/problems/find-the-number-of-winning-players/</a></p> <br><p>给你一个整数 <code>n</code> ，表示在一个游戏中的玩家数目。同时给你一个二维整数数组 <code>pick</code> ，其中 <code>pick[i] = [x<sub>i</sub>, y<sub>i</sub>]</code> 表示玩家 <code>x<sub>i</sub></code> 获得了一个颜色为 <code>y<sub>i</sub></code> 的球。</p> <br><p>如果玩家 <code>i</code> 获得的球中任何一种颜色球的数目 <strong>严格大于</strong> <code>i</code> 个，那么我们说玩家 <code>i</code> 是胜利玩家。换句话说：</p> <br><ul><li>如果玩家 0 获得了任何的球，那么玩家 0 是胜利玩家。</li><li>如果玩家 1 获得了至少 2 个相同颜色的球，那么玩家 1 是胜利玩家。</li><li>...</li><li>如果玩家 <code>i</code> 获得了至少 <code>i + 1</code> 个相同颜色的球，那么玩家 <code>i</code> 是胜利玩家。</li></ul> <br><p>请你返回游戏中 <strong>胜利玩家</strong> 的数目。</p> <br><p><strong>注意</strong>，可能有多个玩家是胜利玩家。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>n = 4, pick = [[0,0],[1,0],[1,0],[2,1],[2,1],[2,0]]</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>2</span></p> <br><p><strong>解释：</strong></p> <br><p>玩家 0 和玩家 1 是胜利玩家，玩家 2 和玩家 3 不是胜利玩家。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>n = 5, pick = [[1,1],[1,2],[1,3],[1,4]]</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>0</span></p> <br><p><strong>解释：</strong></p> <br><p>没有胜利玩家。</p> <br><p><strong class="example">示例 3：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>n = 5, pick = [[1,1],[2,4],[2,4],[2,4]]</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>1</span></p> <br><p><b>解释：</b></p> <br><p>玩家 2 是胜利玩家，因为玩家 2 获得了 3 个颜色为 4 的球。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>2 <= n <= 10</code></li><li><code>1 <= pick.length <= 100</code></li><li><code>pick[i].length == 2</code></li><li><code>0 <= x<sub>i</sub> <= n - 1 </code></li><li><code>0 <= y<sub>i</sub> <= 10</code></li></ul> <br><h3><a id="_72"></a>解题方法：(哈希表)计数</h3> <br><p>使用一个哈希表（数组也可以）记录每个玩家每种颜色的球的获得个数，遍历一遍<code>pick</code>数组即可完成统计。</p> <br><p>然后对于每个玩家，从<code>0</code>到<code>10</code>遍历每种颜色的球的数量，一旦大于玩家编号，就答案加一且快进到下一个玩家。</p> <br><ul><li>时间复杂度<span class="katex--inline"><span class="katex"><span class="katex-mathml"> <br>      <br>
