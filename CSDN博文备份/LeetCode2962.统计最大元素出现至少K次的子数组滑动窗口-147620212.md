<h2><a id="LetMeFly2962_K__1"></a>【LetMeFly】2962.统计最大元素出现至少 K 次的子数组：滑动窗口</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/count-subarrays-where-max-element-appears-at-least-k-times/" rel="nofollow">https://leetcode.cn/problems/count-subarrays-where-max-element-appears-at-least-k-times/</a></p> <br><p>给你一个整数数组 <code>nums</code> 和一个 <strong>正整数</strong> <code>k</code> 。</p> <br><p>请你统计有多少满足 「 <code>nums</code> 中的 <strong>最大</strong> 元素」至少出现 <code>k</code> 次的子数组，并返回满足这一条件的子数组的数目。</p> <br><p>子数组是数组中的一个连续元素序列。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><pre><strong>输入：</strong>nums = [1,3,2,3,3], k = 2<br><strong>输出：</strong>6<br><strong>解释：</strong>包含元素 3 至少 2 次的子数组为：[1,3,2,3]、[1,3,2,3,3]、[3,2,3]、[3,2,3,3]、[2,3,3] 和 [3,3] 。<br></pre> <br><p><strong class="example">示例 2：</strong></p> <br><pre><strong>输入：</strong>nums = [1,4,2,1], k = 3<br><strong>输出：</strong>0<br><strong>解释：</strong>没有子数组包含元素 4 至少 3 次。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= nums.length <= 10<sup>5</sup></code></li><li><code>1 <= nums[i] <= 10<sup>6</sup></code></li><li><code>1 <= k <= 10<sup>5</sup></code></li></ul> <br><h3><a id="_41"></a>解题方法：滑动窗口</h3> <br><p>首先遍历一遍数组得到数组中最大的元素。</p> <br><p>之后使用两个指针分别指向窗口中的第一个和最后一个元素的下标，不断右移右指针，并将新元素加入窗口。</p>
