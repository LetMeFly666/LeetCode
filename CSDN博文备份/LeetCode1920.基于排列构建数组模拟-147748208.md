<h2><a id="LetMeFly1920_1"></a>【LetMeFly】1920.基于排列构建数组</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/build-array-from-permutation/" rel="nofollow">https://leetcode.cn/problems/build-array-from-permutation/</a></p> <br><p>给你一个 <strong>从 0 开始的排列</strong> <code>nums</code>（<strong>下标也从 0 开始</strong>）。请你构建一个 <strong>同样长度</strong> 的数组 <code>ans</code> ，其中，对于每个 <code>i</code>（<code>0 <= i < nums.length</code>），都满足 <code>ans[i] = nums[nums[i]]</code> 。返回构建好的数组 <code>ans</code> 。</p> <br><p><strong>从 0 开始的排列</strong> <code>nums</code> 是一个由 <code>0</code> 到 <code>nums.length - 1</code>（<code>0</code> 和 <code>nums.length - 1</code> 也包含在内）的不同整数组成的数组。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><strong>输入：</strong>nums = [0,2,1,5,3,4]<br><strong>输出：</strong>[0,1,2,4,5,3]<strong><br>解释：</strong>数组 ans 构建如下：<br>ans = [nums[nums[0]], nums[nums[1]], nums[nums[2]], nums[nums[3]], nums[nums[4]], nums[nums[5]]]<br>    = [nums[0], nums[2], nums[1], nums[5], nums[3], nums[4]]<br>    = [0,1,2,4,5,3]</pre> <br><p><strong>示例 2：</strong></p> <br><pre><strong>输入：</strong>nums = [5,0,1,2,3,4]<br><strong>输出：</strong>[4,5,0,1,2,3]<br><strong>解释：</strong>数组 ans 构建如下：<br>ans = [nums[nums[0]], nums[nums[1]], nums[nums[2]], nums[nums[3]], nums[nums[4]], nums[nums[5]]]<br>    = [nums[5], nums[0], nums[1], nums[2], nums[3], nums[4]]<br>    = [4,5,0,1,2,3]</pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= nums.length <= 1000</code></li><li><code>0 <= nums[i] < nums.length</code></li><li><code>nums</code> 中的元素 <strong>互不相同</strong></li></ul> <br><p> </p> <br><p><strong>进阶：</strong>你能在不使用额外空间的情况下解决此问题吗（即 <code>O(1)</code> 内存）？</p> <br><h3><a
