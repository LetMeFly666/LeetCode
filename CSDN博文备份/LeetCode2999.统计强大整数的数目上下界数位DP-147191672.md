<h2><a id="LetMeFly2999DP_1"></a>【LetMeFly】2999.统计强大整数的数目：上下界数位DP</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/count-the-number-of-powerful-integers/" rel="nofollow">https://leetcode.cn/problems/count-the-number-of-powerful-integers/</a></p> <br><p>给你三个整数 <code>start</code> ，<code>finish</code> 和 <code>limit</code> 。同时给你一个下标从 <strong>0</strong> 开始的字符串 <code>s</code> ，表示一个 <strong>正</strong> 整数。</p> <br><p>如果一个 <strong>正</strong> 整数 <code>x</code> 末尾部分是 <code>s</code> （换句话说，<code>s</code> 是 <code>x</code> 的 <strong>后缀</strong>），且 <code>x</code> 中的每个数位至多是 <code>limit</code> ，那么我们称 <code>x</code> 是 <strong>强大的</strong> 。</p> <br><p>请你返回区间 <code>[start..finish]</code> 内强大整数的 <strong>总数目</strong> 。</p> <br><p>如果一个字符串 <code>x</code> 是 <code>y</code> 中某个下标开始（<strong>包括</strong> <code>0</code> ），到下标为 <code>y.length - 1</code> 结束的子字符串，那么我们称 <code>x</code> 是 <code>y</code> 的一个后缀。比方说，<code>25</code> 是 <code>5125</code> 的一个后缀，但不是 <code>512</code> 的后缀。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><pre><b>输入：</b>start = 1, finish = 6000, limit = 4, s = "124"<br><b>输出：</b>5<br><b>解释：</b>区间 [1..6000] 内的强大数字为 124 ，1124 ，2124 ，3124 和 4124 。这些整数的各个数位都 <= 4 且 "124" 是它们的后缀。注意 5124 不是强大整数，因为第一个数位 5 大于 4 。<br>这个区间内总共只有这 5 个强大整数。<br></pre> <br><p><strong class="example">示例 2：</strong></p> <br><pre><b>输入：</b>start = 15, finish = 215, limit = 6, s = "10"<br><b>输出：</b>2<br><b>解释：</b>区间 [15..215] 内的强大整数为 110 和 210 。这些整数的各个数位都 <= 6 且 "10" 是它们的后缀。<br>这个区间总共只有这 2 个强大整数。<br></pre> <br><p><strong class="example">示例 3：</strong></p> <br><pre><b>输入：</b>start = 1000, finish = 2000, limit = 4, s = "3000"<br><b>输出：</b>0<br><b>解释：</b>区间 [1000..2000] 内的整数都小于 3000 ，所以 "3000" 不可能是这个区间内任何整数的后缀。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= start <= finish <= 10<sup>15</sup></code></li><li><code>1 <= limit <= 9</code></li><li><code>1 <= s.length <= floor(log<sub>10</sub>(finish)) + 1</code></li><li><code>s</code> 数位中每个数字都小于等于 <code>limit</code> 。</li><li><code>s</code> 不包含任何前导 0 。</li></ul> <br><h3><a id="DP_54"></a>解题方法：上下界数位DP</h3> <br><p>我们可以从前往后一位一位地构造。在构造的过程中，要保证构造出来的数在<code>start</code>和<code>finish</code>之间（先不考虑<code>limit</code>和后缀<code>s</code>）。</p> <br><blockquote> <br> <p>例如<code>start = 123</code>，<code>finish = 456</code>的话：</p> <br> <ul><li>假如我们构造的整数第一位为<code>1</code>，那么第二位最小为<code>2</code>（可以为<code>2</code>、<code>3</code>、<code>4</code>、<code>...</code>），但不能为<code>0</code>和<code>1</code>。</li><li>假如我们构造的整数第一位为<code>2</code>，那么第二位可以取<code>0 - 9</code>的任意数。</li><li>假设我们构造的整数第一位为<code>3</code>，那么第二位可以取<code>0 - 9</code>的任意数。</li><li>假设我们构造的整数第一位为<code>4</code>，那么第二位最大为<code>5</code>（可以为<code>...</code>、<code>3</code>、<code>4</code>、<code>5</code>），但不能为<code>6 - 9</code>。</li></ul> <br> <p>也就是说，我们可以使用一个函数<code>dfs(i, limitLow, limitHigh)</code>来表示：(当前应该构造第<code>i</code>位、这一位的最小值是否需要限制为<code>start[i]</code>、这一位的最大值是否需要限制为<code>finish[i]</code>)。</p> <br> <ul><li>在构造下一位时，如果<code>limitLow</code>为<code>true</code>且<code>构造的这一位</code>为<code>start[i]</code>，则构造下一位时<code>limitLow</code>仍需为<code>true</code>。</li><li>在构造下一位时，如果<code>limitHigh</code>为<code>true</code>且<code>构造的这一位</code>为<code>finish[i]</code>，则构造下一位时<code>limitHigh</code>仍需为<code>true</code>。</li></ul> <br></blockquote> <br><p>然后就是考虑一下题目的额外两个限制<code>limit</code>和<code>s</code>。</p> <br><blockquote> <br> <p>构造过程中，任何一个元素不能超过<code>limit</code>；</p> <br> <p>构造过程中，若在构造最后的<code>len(s)</code>个元素，则当前元素最多有一种选择<code>s[对应下标]</code>。</p> <br></blockquote> <br><p>最后就是需要记忆化一下。</p> <br><blockquote> <br> <p>记忆化的时候，记录<code>(i, limitLow, limitHigh)</code>三个变量可能有些麻烦，我们也可以只记录<code>i</code>这一个变量，并且在<code>limitLow</code>和<code>limitHigh</code>都为<code>false</code>时再使用记忆化。（因为有<code>true</code>的话不会被再次调用）</p> <br></blockquote>
