<h2><a id="LetMeFly1861_1"></a>【LetMeFly】1861.旋转盒子：模拟(双指针)</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/rotating-the-box/" rel="nofollow">https://leetcode.cn/problems/rotating-the-box/</a></p> <br><p>给你一个 <code>m x n</code> 的字符矩阵 <code>boxGrid</code> ，它表示一个箱子的侧视图。箱子的每一个格子可能为：</p> <br><ul><li><code>'#'</code> 表示石头</li><li><code>'*'</code> 表示固定的障碍物</li><li><code>'.'</code> 表示空位置</li></ul> <br><p>这个箱子被 <strong>顺时针旋转 90 度</strong> ，由于重力原因，部分石头的位置会发生改变。每个石头会垂直掉落，直到它遇到障碍物，另一个石头或者箱子的底部。重力 <strong>不会</strong> 影响障碍物的位置，同时箱子旋转不会产生<strong>惯性</strong> ，也就是说石头的水平位置不会发生改变。</p> <br><p>题目保证初始时 <code>boxGrid</code> 中的石头要么在一个障碍物上，要么在另一个石头上，要么在箱子的底部。</p> <br><p>请你返回一个<em> </em><code>n x m</code> 的矩阵，表示按照上述旋转后，箱子内的结果。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><p><img ["#","#","*","."]]<br><b>输出：</b>[["#","."],<br>      ["#","#"],<br>      ["*","*"],<br>      [".","."]]<br></pre> <br><p><strong>示例 3：</strong></p> <br><p><img alt="" <br><p><strong>提示：</strong></p> <br><ul><li><code>m == boxGrid.length</code></li><li><code>n == boxGrid[i].length</code></li><li><code>1 <= m, n <= 500</code></li><li><code>boxGrid[i][j]</code> 只可能是 <code>'#'</code> ，<code>'*'</code> 或者 <code>'.'</code> 。</li></ul> <br><h3><a id="_74"></a>解题方法：先向右，后旋转</h3> <br><p>当右指针遇到障碍或移出盒子，则从新分配这片连通区域的石头(先数个空格子、后数个石头)。</p> <br><p><strong>矩阵旋转</strong></p> <br><p>令<span class="katex--inline"><span class="katex"><span class="katex-mathml"><br>    <br>     <br>
