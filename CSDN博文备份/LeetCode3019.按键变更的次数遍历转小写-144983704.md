<h2><a id="LetMeFly3019_1"></a>【LetMeFly】3019.按键变更的次数：遍历（转小写）</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/number-of-changing-keys/" rel="nofollow">https://leetcode.cn/problems/number-of-changing-keys/</a></p> <br><p>给你一个下标从<strong> 0</strong> 开始的字符串 <code>s</code> ，该字符串由用户输入。按键变更的定义是：使用与上次使用的按键不同的键。例如 <code>s = "ab"</code> 表示按键变更一次，而 <code>s = "bBBb"</code> 不存在按键变更。</p> <br><p>返回用户输入过程中按键变更的次数。</p> <br><p><strong>注意：</strong><code>shift</code> 或 <code>caps lock</code> 等修饰键不计入按键变更，也就是说，如果用户先输入字母 <code>'a'</code> 然后输入字母 <code>'A'</code> ，不算作按键变更。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><pre><strong>输入：</strong>s = "aAbBcC"<br><strong>输出：</strong>2<br><strong>解释：</strong> <br>从 s[0] = 'a' 到 s[1] = 'A'，不存在按键变更，因为不计入 caps lock 或 shift 。<br>从 s[1] = 'A' 到 s[2] = 'b'，按键变更。<br>从 s[2] = 'b' 到 s[3] = 'B'，不存在按键变更，因为不计入 caps lock 或 shift 。<br>从 s[3] = 'B' 到 s[4] = 'c'，按键变更。<br>从 s[4] = 'c' 到 s[5] = 'C'，不存在按键变更，因为不计入 caps lock 或 shift 。<br></pre> <br><p><strong class="example">示例 2：</strong></p> <br><pre><strong>输入：</strong>s = "AaAaAaaA"<br><strong>输出：</strong>0<br><strong>解释：</strong> 不存在按键变更，因为这个过程中只按下字母 'a' 和 'A' ，不需要进行按键变更。<br> <br> <br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= s.length <= 100</code></li><li><code>s</code> 仅由英文大写字母和小写字母组成。</li></ul> <br><h3><a id="_45"></a>解题方法：遍历</h3> <br><p>从第二个字符开始遍历字符串，如果当前字符串的小写和前一个字符的小写不相同，则答案数量加一。</p> <br><ul><li>时间复杂度<span class="katex--inline"><span class="katex"><span
