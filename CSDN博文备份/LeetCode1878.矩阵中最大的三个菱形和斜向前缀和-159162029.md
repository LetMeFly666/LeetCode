<h2><a id="LetMeFly1878_1"></a>【LetMeFly】1878.矩阵中最大的三个菱形和：斜向前缀和</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/get-biggest-three-rhombus-sums-in-a-grid/" rel="nofollow">https://leetcode.cn/problems/get-biggest-three-rhombus-sums-in-a-grid/</a></p> <br><p>给你一个 <code>m x n</code> 的整数矩阵 <code>grid</code> 。</p> <br><p><strong>菱形和</strong> 指的是 <code>grid</code> 中一个正菱形 <strong>边界</strong> 上的元素之和。本题中的菱形必须为正方形旋转45度，且四个角都在一个格子当中。下图是四个可行的菱形，每个菱形和应该包含的格子都用了相应颜色标注在图中。</p> <br><img alt="" src="https://i-blog.csdnimg.cn/img_convert/63f098790ce66a9b506ba08ec10bd1ba.png" /> <br><p> </p> <br><p>注意，菱形可以是一个面积为 0 的区域，如上图中右下角的紫色菱形所示。</p> <br><p>请你按照 <strong>降序</strong> 返回 <code>grid</code> 中三个最大的 <strong>互不相同的菱形和</strong> 。如果不同的和少于三个，则将它们全部返回。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><img alt="" src="https://i-blog.csdnimg.cn/img_convert/48f018b4ec18d3367f81dd62433bd522.png" /> <br><pre><b>输入：</b>grid = [[3,4,5,1,3],[3,3,4,2,3],[20,30,200,40,10],[1,5,5,4,1],[4,3,2,2,5]]<br><b>输出：</b>[228,216,211]<br><b>解释：</b>最大的三个菱形和如上图所示。<br>- 蓝色：20 + 3 + 200 + 5 = 228<br>- 红色：200 + 2 + 10 + 4 = 216<br>- 绿色：5 + 200 + 4 + 2 = 211<br></pre> <br><p><strong>示例 2：</strong></p> <br><img alt="" src="https://i-blog.csdnimg.cn/img_convert/a28edb6d58a7e687151f3e1404e0f25a.png" /> <br><pre><b>输入：</b>grid = [[1,2,3],[4,5,6],[7,8,9]]<br><b>输出：</b>[20,9,8]<br><b>解释：</b>最大的三个菱形和如上图所示。<br>- 蓝色：4 + 2 + 6 + 8 = 20<br>- 红色：9 （右下角红色的面积为 0 的菱形）<br>- 绿色：8 （下方中央面积为 0 的菱形）<br></pre> <br><p><strong>示例 3：</strong></p> <br><pre><b>输入：</b>grid = [[7,7,7]]<br><b>输出：</b>[7]<br><b>解释：</b>所有三个可能的菱形和都相同，所以返回 [7] 。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>m == grid.length</code></li><li><code>n == grid[i].length</code></li><li><code>1 <= m, n <= 100</code></li><li><code>1 <= grid[i][j] <= 10<sup>5</sup></code></li></ul> <br><h3><a
