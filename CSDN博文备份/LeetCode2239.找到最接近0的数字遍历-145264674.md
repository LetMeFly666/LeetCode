<h2><a id="LetMeFly2239_0__1"></a>【LetMeFly】2239.找到最接近 0 的数字：遍历</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/find-closest-number-to-zero/" rel="nofollow">https://leetcode.cn/problems/find-closest-number-to-zero/</a></p> <br><p>给你一个长度为 <code>n</code> 的整数数组 <code>nums</code> ，请你返回 <code>nums</code> 中最 <strong>接近</strong> <code>0</code> 的数字。如果有多个答案，请你返回它们中的 <strong>最大值</strong> 。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><b>输入：</b>nums = [-4,-2,1,4,8]<br><b>输出：</b>1<br><strong>解释：</strong><br>-4 到 0 的距离为 |-4| = 4 。<br>-2 到 0 的距离为 |-2| = 2 。<br>1 到 0 的距离为 |1| = 1 。<br>4 到 0 的距离为 |4| = 4 。<br>8 到 0 的距离为 |8| = 8 。<br>所以，数组中距离 0 最近的数字为 1 。<br></pre> <br><p><strong>示例 2：</strong></p> <br><pre><b>输入：</b>nums = [2,-1,1]<br><b>输出：</b>1<br><b>解释：</b>1 和 -1 都是距离 0 最近的数字，所以返回较大值 1 。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= n <= 1000</code></li><li><code>-10<sup>5</sup> <= nums[i] <= 10<sup>5</sup></code></li></ul> <br><h3><a id="_40"></a>解题方法：遍历</h3> <br><ol><li><code>abs(t) < abs(ans)</code></li><li><code>abs(t) == abs(ans)</code>且<code>ans < 0</code></li></ol> <br><p>最终返回<code>ans</code>。</p> <br><ul><li>时间复杂度<span class="katex--inline"><span
