<h2><a id="LetMeFly966_1"></a>【LetMeFly】966.元音拼写检查器：三个哈希表实现</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/vowel-spellchecker/" rel="nofollow">https://leetcode.cn/problems/vowel-spellchecker/</a></p> <br><p>在给定单词列表 <code>wordlist</code> 的情况下，我们希望实现一个拼写检查器，将查询单词转换为正确的单词。</p> <br><p>对于给定的查询单词 <code>query</code>，拼写检查器将会处理两类拼写错误：</p> <br><ul><li>大小写：如果查询匹配单词列表中的某个单词（<strong>不区分大小写</strong>），则返回的正确单词与单词列表中的大小写相同。 <pre><code><ul><br>	<li>例如：<code>wordlist = ["yellow"]</code>, <code>query = "YellOw"</code>: <code>correct = "yellow"</code></li><br>	<li>例如：<code>wordlist = ["Yellow"]</code>, <code>query = "yellow"</code>: <code>correct = "Yellow"</code></li><br>	<li>例如：<code>wordlist = ["yellow"]</code>, <code>query = "yellow"</code>: <code>correct = "yellow"</code></li><br></ul><br></li><br><li>元音错误：如果在将查询单词中的元音 <code>('a', 'e', 'i', 'o', 'u')</code>&nbsp;&nbsp;分别替换为任何元音后，能与单词列表中的单词匹配（<strong>不区分大小写</strong>），则返回的正确单词与单词列表中的匹配项大小写相同。<br><ul><br>	<li>例如：<code>wordlist = ["YellOw"]</code>, <code>query = "yollow"</code>: <code>correct = "YellOw"</code></li><br>	<li>例如：<code>wordlist = ["YellOw"]</code>, <code>query = "yeellow"</code>: <code>correct = ""</code> （无匹配项）</li><br>	<li>例如：<code>wordlist = ["YellOw"]</code>, <code>query = "yllw"</code>: <code>correct = ""</code> （无匹配项）</li><br></ul><br></li><br></code></pre> </li></ul> <br><p>此外，拼写检查器还按照以下优先级规则操作：</p> <br><ul><li>当查询完全匹配单词列表中的某个单词（<strong>区分大小写</strong>）时，应返回相同的单词。</li><li>当查询匹配到大小写问题的单词时，您应该返回单词列表中的第一个这样的匹配项。</li><li>当查询匹配到元音错误的单词时，您应该返回单词列表中的第一个这样的匹配项。</li><li>如果该查询在单词列表中没有匹配项，则应返回空字符串。</li></ul> <br><p>给出一些查询 <code>queries</code>，返回一个单词列表 <code>answer</code>，其中 <code>answer[i]</code> 是由查询 <code>query = queries[i]</code> 得到的正确单词。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><strong>输入：</strong>wordlist = ["KiTe","kite","hare","Hare"], queries = ["kite","Kite","KiTe","Hare","HARE","Hear","hear","keti","keet","keto"]<br><strong>输出：</strong>["kite","KiTe","KiTe","Hare","hare","","","KiTe","","KiTe"]</pre> <br><p><strong>示例 2:</strong></p> <br><pre><b>输入：</b>wordlist = ["yellow"], queries = ["YellOw"]<br><b>输出：</b>["yellow"]<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= wordlist.length, queries.length <= 5000</code></li><li><code>1 <= wordlist[i].length, queries[i].length <= 7</code></li><li><code>wordlist[i]</code> 和 <code>queries[i]</code> 只包含英文字母</li></ul> <br><h3><a id="_65"></a>解题方法：哈希表</h3> <br><p>第一优先级：字符串完全相同。我们只需要使用一个哈希表把原始字符加入哈希表中，对于一个query就能知道是否存在完全匹配的word了。若存在则返回，不存在进入第二优先级。</p> <br><p>第二优先级：字符串忽略大小写看是否相同。我们只需要将每个字符串全转为小写字母后插入哈希表即可。键为小写字符串，值为第一个对应这个小写字符串的原始字符串。对于一个query，小写字符化后看是否在哈希表中，若在则返回否则进入第三优先级。</p> <br><p>第三优先级：字符串忽略大小写且可以自由替换元音音符。将字符串小写后并将所有元音音符替换为’a’并插入哈希表中。插入方式和query方式同上。</p> <br><p>第四优先级：直接返回空字符串。</p> <br><ul><li>时间复杂度<span class="katex--inline"><span class="katex"><span
