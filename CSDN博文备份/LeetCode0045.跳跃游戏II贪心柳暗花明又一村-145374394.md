<h2><a id="LetMeFly45_II_1"></a>【LetMeFly】45.跳跃游戏 II：贪心（柳暗花明又一村）</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/jump-game-ii/" rel="nofollow">https://leetcode.cn/problems/jump-game-ii/</a></p> <br><p>给定一个长度为 <code>n</code> 的 <strong>0 索引</strong>整数数组 <code>nums</code>。初始位置为 <code>nums[0]</code>。</p> <br><p>每个元素 <code>nums[i]</code> 表示从索引 <code>i</code> 向前跳转的最大长度。换句话说，如果你在 <code>nums[i]</code> 处，你可以跳转到任意 <code>nums[i + j]</code> 处:</p> <br><ul><li><code>0 <= j <= nums[i]</code> </li><li><code>i + j < n</code></li></ul> <br><p>返回到达 <code>nums[n - 1]</code> 的最小跳跃次数。生成的测试用例可以到达 <code>nums[n - 1]</code>。</p> <br><p> </p> <br><p><strong>示例 1:</strong></p> <br><pre><strong>输入:</strong> nums = [2,3,1,1,4]<br><strong>输出:</strong> 2<br><strong>解释:</strong> 跳到最后一个位置的最小跳跃数是 <code>2</code>。<br>     从下标为 0 跳到下标为 1 的位置，跳 <code>1</code> 步，然后跳 <code>3</code> 步到达数组的最后一个位置。<br></pre> <br><p><strong>示例 2:</strong></p> <br><pre><strong>输入:</strong> nums = [2,3,0,1,4]<br><strong>输出:</strong> 2<br></pre> <br><p> </p> <br><p><strong>提示:</strong></p> <br><ul><li><code>1 <= nums.length <= 10<sup>4</sup></code></li><li><code>0 <= nums[i] <= 1000</code></li><li>题目保证可以到达 <code>nums[n-1]</code></li></ul> <br><h3><a id="_46"></a>解题思路</h3> <br><p>假设第一步从<code>0</code>可以跳到<code>1, 2, 3</code>这<code>3</code>个位置，那么第二步从这三个位置中的哪一个开始起跳？</p> <br><blockquote> <br> <p>当然是从哪个能跳更远，就从哪个开始跳。</p> <br></blockquote> <br><p>然后问题就解决了。</p> <br><h3><a id="_54"></a>解题方法：贪心</h3>
