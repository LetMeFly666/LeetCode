<h2><a id="LetMeFly3637_I_1"></a>【LetMeFly】3637.三段式数组 I：一次遍历（三种实现）</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/trionic-array-i/" rel="nofollow">https://leetcode.cn/problems/trionic-array-i/</a></p> <br><p>给你一个长度为 <code>n</code> 的整数数组 <code>nums</code>。</p> <br><p>如果存在索引 <code>0 < p < q < n − 1</code>，使得数组满足以下条件，则称其为 <strong>三段式数组（trionic）</strong>：</p> <br><ul><li><code>nums[0...p]</code> <strong>严格</strong> 递增，</li><li><code>nums[p...q]</code> <strong>严格</strong> 递减，</li><li><code>nums[q...n − 1]</code> <strong>严格</strong> 递增。</li></ul> <br><p>如果 <code>nums</code> 是三段式数组，返回 <code>true</code>；否则，返回 <code>false</code>。</p> <br><p> </p> <br><p><strong class="example">示例 1:</strong></p> <br><div class="example-block"> <br> <p><strong>输入:</strong> <span class="example-io">nums = [1,3,5,4,2,6]</span></p> <br></div><br><p><strong>输出:</strong> <span class="example-io">true</span></p> <br><p><strong>解释:</strong></p> <br><p>选择 <code>p = 2</code>, <code>q = 4</code>：</p> <br><ul><li><code>nums[0...2] = [1, 3, 5]</code> 严格递增 (<code>1 < 3 < 5</code>)。</li><li><code>nums[2...4] = [5, 4, 2]</code> 严格递减 (<code>5 > 4 > 2</code>)。</li><li><code>nums[4...5] = [2, 6]</code> 严格递增 (<code>2 < 6</code>)。</li></ul> <br><p><strong class="example">示例 2:</strong></p> <br><div class="example-block"> <br> <p><strong>输入:</strong> <span class="example-io">nums = [2,1,3]</span></p> <br></div><br><p><strong>输出:</strong> <span class="example-io">false</span></p> <br><p><strong>解释:</strong></p> <br><p>无法选出能使数组满足三段式要求的 <code>p</code> 和 <code>q</code> 。</p> <br><p> </p> <br><p><strong>提示:</strong></p> <br><ul><li><code>3 <= n <= 100</code></li><li><code>-1000 <= nums[i] <= 1000</code></li></ul> <br><h3><a id="_60"></a>解题方法：遍历</h3> <br><p>使用一个状态<span class="katex--inline"><span class="katex"><span class="katex-mathml"><br>    <br>     <br>      <br>       <br>
