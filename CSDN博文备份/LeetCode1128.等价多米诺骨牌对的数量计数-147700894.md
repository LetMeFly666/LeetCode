<h2><a id="LetMeFly1128_1"></a>【LetMeFly】1128.等价多米诺骨牌对的数量：计数</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/number-of-equivalent-domino-pairs/" rel="nofollow">https://leetcode.cn/problems/number-of-equivalent-domino-pairs/</a></p> <br><p>给你一组多米诺骨牌 <code>dominoes</code> 。</p> <br><p>形式上，<code>dominoes[i] = [a, b]</code> 与 <code>dominoes[j] = [c, d]</code> <strong>等价</strong> 当且仅当 (<code>a == c</code> 且 <code>b == d</code>) 或者 (<code>a == d</code> 且 <code>b == c</code>) 。即一张骨牌可以通过旋转 <code>0</code> 度或 <code>180</code> 度得到另一张多米诺骨牌。</p> <br><p>在 <code>0 <= i < j < dominoes.length</code> 的前提下，找出满足 <code>dominoes[i]</code> 和 <code>dominoes[j]</code> 等价的骨牌对 <code>(i, j)</code> 的数量。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> [[1,2],[1,2],[1,1],[1,2],[2,2]]<br><strong>输出：</strong>3<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= dominoes.length <= 4 * 10<sup>4</sup></code></li><li><code>dominoes[i].length == 2</code></li><li><code>1 <= dominoes[i][j] <= 9</code></li></ul> <br><h3><a id="_39"></a>解题方法：计数</h3>
