<strong>从 0 开始</strong> 计数。</p> <br><p>下标对 <code>(i, j)</code> 中 <code>0 <= i < nums1.length</code> 且 <code>0 <= j < nums2.length</code> 。如果该下标对同时满足 <code>i <= j</code> 且 <code>nums1[i] <= 数组。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><strong>输入：</strong>nums1 = [55,30,5,4,2], nums2 = [100,20,10,10,5]<br><strong>输出：</strong>2<br><strong>解释：</strong>有效下标对是 (0,0), (2,2), [10,10,1]<br><strong>输出：</strong>1<br><strong>解释：</strong>有效下标对是 (0,0), (0,1) 和 (1,1) 。<br>最大距离是 1 ，对应下标对 (0,1) 。</pre> <br><p><strong>示例 3：</strong></p> <br><pre><strong>输入：</strong>nums1 = [30,29,19,5], nums2 = [25,25,25,25,25]<br><strong>输出：</strong>2<br><strong>解释：</strong>有效下标对是 (2,2), (2,3), (2,4), (3,3) 和 (3,4) 。<br>最大距离是 2 ，对应下标对 (2,4) 。<br></pre> <br><p> </p>
