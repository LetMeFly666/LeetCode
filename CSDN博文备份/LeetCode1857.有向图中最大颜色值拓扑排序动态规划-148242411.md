<h2><a id="LetMeFly1857_1"></a>【LetMeFly】1857.有向图中最大颜色值：拓扑排序+动态规划</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/largest-color-value-in-a-directed-graph/" rel="nofollow">https://leetcode.cn/problems/largest-color-value-in-a-directed-graph/</a></p> <br><p>给你一个 <strong>有向图</strong> ，它含有 <code>n</code> 个节点和 <code>m</code> 条边。节点编号从 <code>0</code> 到 <code>n - 1</code> 。</p> <br><p>给你一个字符串 <code>colors</code> ，其中 <code>colors[i]</code> 是小写英文字母，表示图中第 <code>i</code> 个节点的 <b>颜色</b> （下标从 <strong>0</strong> 开始）。同时给你一个二维数组 <code>edges</code> ，其中 <code>edges[j] = [a<sub>j</sub>, b<sub>j</sub>]</code> 表示从节点 <code>a<sub>j</sub></code> 到节点 <code>b<sub>j</sub></code><sub> </sub>有一条 <strong>有向边</strong> 。</p> <br><p>图中一条有效 <strong>路径</strong> 是一个点序列 <code>x<sub>1</sub> -> x<sub>2</sub> -> x<sub>3</sub> -> ... -> x<sub>k</sub></code> ，对于所有 <code>1 <= i < k</code> ，从 <code>x<sub>i</sub></code> 到 <code>x<sub>i+1</sub></code> 在图中有一条有向边。路径的 <strong>颜色值</strong> 是路径中 <strong>出现次数最多</strong> 颜色的节点数目。</p> <br><p>请你返回给定图中有效路径里面的 <strong>最大颜色值</strong><strong> 。</strong>如果图中含有环，请返回 <code>-1</code> 。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/229d4ff7c0ab5b856bc614ac5437f58d.png" /></p> [[0,0]]<br><b>输出：</b>-1<br><b>解释：</b>从 0 到 0 有一个环。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>n == colors.length</code></li><li><code>m == n</code></li></ul> <br><h3><a id="_48"></a>解题方法：拓扑排序+动态规划</h3> <br><p>记录每个节点的入度(入度为指向这个节点的边的个数)，记录每个节点指向哪些节点。</p> <br><p>创建一个队列，一旦有节点入度为0则入队（拓扑排序）。</p> <br><p>创建一个DP数组，DP[i][j]表示从路径起点到节点i颜色j的最大个数。</p> dp[thisNode][j])。</p> <br></blockquote> <br><p>最终，若仍有节点入度非零，则代表有环，返回-1；否则，返回dp[i][j]中最大的一个。</p> <br><ul><li>时间复杂度<span class="katex--inline"><span class="katex"><span class="katex-mathml"> <br>
