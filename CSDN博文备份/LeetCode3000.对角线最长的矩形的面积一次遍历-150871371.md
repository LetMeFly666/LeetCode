<h2><a id="LetMeFly3000_1"></a>【LetMeFly】3000.对角线最长的矩形的面积：一次遍历</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/maximum-area-of-longest-diagonal-rectangle/" rel="nofollow">https://leetcode.cn/problems/maximum-area-of-longest-diagonal-rectangle/</a></p> <br><p>给你一个下标从<strong> 0</strong> 开始的二维整数数组 <code>dimensions</code>。</p> <br><p>对于所有下标 <code>i</code>（<code>0 <= i < dimensions.length</code>），<code>dimensions[i][0]</code> 表示矩形  <code>i</code> 的长度，而 <code>dimensions[i][1]</code> 表示矩形  <code>i</code> 的宽度。</p> <br><p>返回对角线最 <strong>长 </strong>的矩形的<strong> 面积 </strong>。如果存在多个对角线长度相同的矩形，返回面积最<strong> 大 </strong>的矩形的面积。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><pre><strong>输入：</strong>dimensions = [[9,3],[8,6]]<br><strong>输出：</strong>48<br><strong>解释：</strong><br>下标 = 0，长度 = 9，宽度 = 3。对角线长度 = sqrt(9 * 9 + 3 * 3) = sqrt(90) ≈<br>  9.487。<br>下标 = 1，长度 = 8，宽度 = 6。对角线长度 = sqrt(8 * 8 + 6 * 6) = sqrt(100) = 10。<br>因此，下标为 1 的矩形对角线更长，所以返回面积 = 8 * 6 = 48。<br></pre> <br><p><strong class="example">示例 2：</strong></p> <br><pre><strong>输入：</strong>dimensions = [[3,4],[4,3]]<br><strong>输出：</strong>12<br><strong>解释：</strong>两个矩形的对角线长度相同，为 5，所以最大面积 = 12。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= dimensions.length <= 100</code></li><li><code>dimensions[i].length == 2</code></li><li><code>1 <= dimensions[i][0], dimensions[i][1] <= 100</code></li></ul> <br><h3><a id="_44"></a>解题方法：遍历维护最大值</h3> <br><p>使用一个变量l2维护最大的对角线长度的平方（避免无意义的开根号），遍历所有矩形：</p> <br><ul><li>如果当前矩形对角线长度的平方大于l2，则更新l2并直接更新答案为当前矩形的面积；</li><li>否则如果当前对角线长度的平方等于l2，则更新答案为当前矩形面积和答案的最大值。</li></ul> <br><h4><a id="_51"></a>时空复杂度分析</h4>
