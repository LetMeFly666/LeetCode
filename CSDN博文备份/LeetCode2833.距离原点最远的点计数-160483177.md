<h2><a id="LetMeFly2833_1"></a>【LetMeFly】2833.距离原点最远的点：计数</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/furthest-point-from-origin/" rel="nofollow">https://leetcode.cn/problems/furthest-point-from-origin/</a></p> <br><p>给你一个长度为 <code>n</code> 的字符串 <code>moves</code> ，该字符串仅由字符 <code>'L'</code>、<code>'R'</code> 和 <code>'_'</code> 组成。字符串表示你在一条原点为 <code>0</code> 的数轴上的若干次移动。</p> <br><p>你的初始位置就在原点（<code>0</code>），第 <code>i</code> 次移动过程中，你可以根据对应字符选择移动方向：</p> <br><ul><li>如果 <code>moves[i] = 'L'</code> 或 <code>moves[i] = '_'</code> ，可以选择向左移动一个单位距离</li><li>如果 <code>moves[i] = 'R'</code> 或 <code>moves[i] = '_'</code> ，可以选择向右移动一个单位距离</li></ul> <br><p>移动 <code>n</code> 次之后，请你找出可以到达的距离原点 <strong>最远</strong> 的点，并返回 <strong>从原点到这一点的距离</strong> 。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><strong>输入：</strong>moves = "L_RL__R"<br><strong>输出：</strong>3<br><strong>解释：</strong>可以到达的距离原点 0 最远的点是 -3 ，移动的序列为 "LLRLLLR" 。<br></pre> <br><p><strong>示例 2：</strong></p> <br><pre><strong>输入：</strong>moves = "_R__LL_"<br><strong>输出：</strong>5<br><strong>解释：</strong>可以到达的距离原点 0 最远的点是 -5 ，移动的序列为 "LRLLLLL" 。<br></pre> <br><p><strong>示例 3：</strong></p> <br><pre><strong>输入：</strong>moves = "_______"<br><strong>输出：</strong>7<br><strong>解释：</strong>可以到达的距离原点 0 最远的点是 7 ，移动的序列为 "RRRRRRR" 。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= moves.length == n <= 50</code></li><li><code>moves</code> 仅由字符 <code>'L'</code>、<code>'R'</code> 和 <code>'_'</code> 组成</li></ul>
