<h2><a id="LetMeFly3159x_1"></a>【LetMeFly】3159.查询数组中元素的出现位置：存x下标</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/find-occurrences-of-an-element-in-an-array/" rel="nofollow">https://leetcode.cn/problems/find-occurrences-of-an-element-in-an-array/</a></p> <br><p>给你一个整数数组 <code>nums</code> ，一个整数数组 <code>queries</code> 和一个整数 <code>x</code> 。</p> <br><p>对于每个查询 <code>queries[i]</code> ，你需要找到 <code>nums</code> 中第 <code>queries[i]</code> 个 <code>x</code> 的位置，并返回它的下标。如果数组中 <code>x</code> 的出现次数少于 <code>queries[i]</code> ，该查询的答案为 -1 。</p> <br><p>请你返回一个整数数组 <code>answer</code> ，包含所有查询的答案。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>nums = [1,3,1,7], queries = [1,3,2,4], x = 1</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>[0,-1,2,-1]</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>第 1 个查询，第一个 1 出现在下标 0 处。</li><li>第 2 个查询，<code>nums</code> 中只有两个 1 ，所以答案为 -1 。</li><li>第 3 个查询，第二个 1 出现在下标 2 处。</li><li>第 4 个查询，<code>nums</code> 中只有两个 1 ，所以答案为 -1 。</li></ul> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><span class="example-io"><b>输入：</b>nums = [1,2,3], queries = [10], x = 5</span></p> <br></div> <br><p><span class="example-io"><b>输出：</b>[-1]</span></p> <br><p><strong>解释：</strong></p> <br><ul><li>第 1 个查询，<code>nums</code> 中没有 5 ，所以答案为 -1 。</li></ul> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= nums.length, queries.length <= 10<sup>5</sup></code></li><li><code>1 <= queries[i] <= 10<sup>5</sup></code></li><li><code>1 <= nums[i], x <= 10<sup>4</sup></code></li></ul>
