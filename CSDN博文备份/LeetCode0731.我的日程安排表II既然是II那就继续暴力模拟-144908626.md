<h2><a id="LetMeFly731_IIII_1"></a>【LetMeFly】731.我的日程安排表 II：既然是II，那就继续暴力模拟</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/my-calendar-ii/" rel="nofollow">https://leetcode.cn/problems/my-calendar-ii/</a></p> <br><p>实现一个程序来存放你的日程安排。如果要添加的时间内不会导致三重预订时，则可以存储这个新的日程安排。</p> <br><p>当三个日程安排有一些时间上的交叉时（例如三个日程安排都在同一时间内），就会产生 <strong>三重预订</strong>。</p> <br><p>事件能够用一对整数 <code>startTime</code> 和 <code>endTime</code> 表示，在一个半开区间的时间 <code>[startTime, endTime)</code> 上预定。实数 <code>x</code> 的范围为  <code>startTime <= x < endTime</code>。</p> <br><p>实现 <code>MyCalendarTwo</code> 类：</p> <br><ul><li><code>MyCalendarTwo()</code> 初始化日历对象。</li><li><code>boolean book(int startTime, int endTime)</code> 如果可以将日程安排成功添加到日历中而不会导致三重预订，返回 <code>true</code>。否则，返回 <code>false</code> 并且不要将该日程安排添加到日历中。</li></ul> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><pre><strong>输入：</strong><br>["MyCalendarTwo", "book", "book", "book", "book", "book", "book"]<br>[[], [10, 20], [50, 60], [10, 40], [5, 15], [5, 10], [25, 55]]<br><strong>输出：</strong><br>[null, true, true, true, false, true, true]<br><br><strong>解释：</strong><br>MyCalendarTwo myCalendarTwo = new MyCalendarTwo();<br>myCalendarTwo.book(10, 20); // 返回 True，能够预定该日程。<br>myCalendarTwo.book(50, 60); // 返回 True，能够预定该日程。<br>myCalendarTwo.book(10, 40); // 返回 True，该日程能够被重复预定。<br>myCalendarTwo.book(5, 15);  // 返回 False，该日程导致了三重预定，所以不能预定。<br>myCalendarTwo.book(5, 10); // 返回 True，能够预定该日程，因为它不使用已经双重预订的时间 10。<br>myCalendarTwo.book(25, 55); // 返回 True，能够预定该日程，因为时间段 [25, 40) 将被第三个日程重复预定，时间段 [40, 50) 将被单独预定，而时间段 [50, 55) 将被第二个日程重复预定。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>0 <= start < end <= 10<sup>9</sup></code></li><li>最多调用 <code>book</code> 1000 次。</li></ul> <br><h3><a id="_50"></a>解题方法：记录二重叠时间，判断是否有交集</h3> <br><p>在<a
