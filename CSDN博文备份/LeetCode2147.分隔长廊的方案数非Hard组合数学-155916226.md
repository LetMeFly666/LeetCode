<h2><a id="LetMeFly2147Hard_1"></a>【LetMeFly】2147.分隔长廊的方案数：非Hard组合数学</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/number-of-ways-to-divide-a-long-corridor/" rel="nofollow">https://leetcode.cn/problems/number-of-ways-to-divide-a-long-corridor/</a></p> <br><p>在一个图书馆的长廊里，有一些座位和装饰植物排成一列。给你一个下标从 <strong>0</strong> 开始，长度为 <code>n</code> 的字符串 <code>corridor</code> ，它包含字母 <code>'S'</code> 和 <code>'P'</code> ，其中每个 <code>'S'</code> 表示一个座位，每个 <code>'P'</code> 表示一株植物。</p> <br><p>在下标 <code>0</code> 的左边和下标 <code>n - 1</code> 的右边 <strong>已经</strong> 分别各放了一个屏风。你还需要额外放置一些屏风。每一个位置 <code>i - 1</code> 和 <code>i</code> 之间（<code>1 <= i <= n - 1</code>），至多能放一个屏风。</p> <br><p>请你将走廊用屏风划分为若干段，且每一段内都 <strong>恰好有两个座位</strong> ，而每一段内植物的数目没有要求。可能有多种划分方案，如果两个方案中有任何一个屏风的位置不同，那么它们被视为 <strong>不同</strong> 方案。</p> <br><p>请你返回划分走廊的方案数。由于答案可能很大，请你返回它对 <code>10<sup>9</sup> + 7</code> <strong>取余</strong> 的结果。如果没有任何方案，请返回 <code>0</code> 。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/a3c8a7e66a387f31ad15be211fe6a444.png" /></p> <br><pre><b>输入：</b>corridor = "SSPPSPS"<br><b>输出：</b>3<br><b>解释：</b>总共有 3 种不同分隔走廊的方案。<br>上图中黑色的竖线表示已经放置好的屏风。<br>上图每种方案中，每一段都恰好有 <strong>两个</strong> 座位。<br></pre> <br><p><strong>示例 2：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/5433e652764bdcd03b769f03880c4f8f.png" /></p> <br><pre><b>输入：</b>corridor = "PPSPSP"<br><b>输出：</b>1<br><b>解释：</b>只有 1 种分隔走廊的方案，就是不放置任何屏风。<br>放置任何的屏风都会导致有一段无法恰好有 2 个座位。<br></pre> <br><p><strong>示例 3：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/3902117e52fc3d2120220aac2e9f2568.png" /></p> <br><pre><b>输入：</b>corridor = "S"<br><b>输出：</b>0<br><b>解释：</b>没有任何方案，因为总是有一段无法恰好有 2 个座位。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>n == corridor.length</code></li><li><code>1 <= n <= 10<sup>5</sup></code></li><li><code>corridor[i]</code> 要么是 <code>'S'</code> ，要么是 <code>'P'</code> 。</li></ul> <br><h3><a id="_57"></a>解题方法：遍历</h3> <br><p>从左往右遍历，每出现总计两个座位就要进行一次分隔，本次分隔方案数为两个座位中后一个座位与<strong>下一个</strong>座位之间的绿植数加一。（若后续再无座位，则不需要放置隔板）</p> <br><p>总方案数就是每次放置隔板时的方案数之积。</p> <br><p>额外注意，本题给定的答案中，若没有座位，则输出0而非“一个隔板都不放的这唯一一种方案”1。</p> <br><h4><a id="_65"></a>具体做法</h4> <br><p>使用一个变量ing记录当前是否处在两个座位之后下一个座位之前的数绿植状态，使用一个变量now记录当前总计座位数或绿植数，遍历时候使用几个if-else就好了。</p> <br><p>额外注意，可以使用一个变量atLeast2记录是否至少有两个座位，遍历过程中一旦出现累计两个座位则将该值赋值为true。</p> <br><p>最终若不是在数绿植状态（不是刚好两个座位后）或一共也没有两个座位，返回0。</p> <br><h4><a id="_73"></a>时空复杂度分析</h4> <br><ul><li>时间复杂度<span class="katex--inline"><span class="katex"><span class="katex-mathml"> <br>      <br>
