<h2><a id="LetMeFly3226O1_1"></a>【LetMeFly】3226.使两个整数相等的位更改次数：位运算（接近O(1)的做法）</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/number-of-bit-changes-to-make-two-integers-equal/" rel="nofollow">https://leetcode.cn/problems/number-of-bit-changes-to-make-two-integers-equal/</a></p> <br><p>给你两个正整数 <code>n</code> 和 <code>k</code>。</p> <br><p>你可以选择 <code>n</code> 的 <strong>二进制表示</strong> 中任意一个值为 1 的位，并将其改为 0。</p> <br><p>返回使得 <code>n</code> 等于 <code>k</code> 所需要的更改次数。如果无法实现，返回 -1。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div <br><p><strong>解释：</strong><br /> 最初，<code>n</code> 和 <code>k</code> 的二进制表示分别为 <code>n = (1101)<sub>2</sub></code> 和 <code>k = (0100)<sub>2</sub></code>，</p> <br><p>我们可以改变 <code>n</code> 的第一位和第四位。结果整数为 <code>n = (<u><strong>0</strong></u>10<u><strong>0</strong></u>)<sub>2</sub> = k</code>。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">n = 21, k = 21</span></p> <br></div> <br><p><strong>输出：</strong> <span class="example-io">0</span></p> <br><p><strong>解释：</strong><br /> <code>n</code> 和 <code>k</code> 已经相等，因此不需要更改。</p> <br><p><strong class="example">示例 3：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">n = 14, k = 13</span></p> <br></div> <br><p><strong>输出：</strong> <span class="example-io">-1</span></p> <br><p><strong>解释：</strong><br /> 无法使 <code>n</code> 等于 <code>k</code>。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= n, k <= 10<sup>6</sup></code></li></ul> <br><h3><a id="_58"></a>解题方法：位运算</h3> <br><p>如何通过位运算判断能否实现？</p> <br><blockquote> <br> <p>若<code>n | k ==
