<h2><a id="LetMeFly2452_1"></a>【LetMeFly】2452.距离字典两次编辑以内的单词：暴力</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/words-within-two-edits-of-dictionary/" rel="nofollow">https://leetcode.cn/problems/words-within-two-edits-of-dictionary/</a></p> <br><p>给你两个字符串数组 <code>queries</code> 和 <code>dictionary</code> 。数组中所有单词都只包含小写英文字母，且长度都相同。</p> <br><p>一次 <strong>编辑</strong> 中，你可以从 <code>queries</code> 中选择一个单词，将任意一个字母修改成任何其他字母。从 <code>queries</code> 中找到所有满足以下条件的字符串：<strong>不超过</strong> 两次编辑内，字符串与 <code>dictionary</code> 中某个字符串相同。</p> <br><p>请你返回<em> </em><code>queries</code> 中的单词列表，这些单词距离 <code>dictionary</code> 中的单词 <strong>编辑次数</strong> 不超过 <strong>两次</strong> 。单词返回的顺序需要与 <code>queries</code> 中原本顺序相同。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><b>输入：</b>queries = ["word","note","ants","wood"], dictionary = ["wood","joke","moat"]<br><b>输出：</b>["word","note","wood"]<br><strong>解释：</strong><br>- 将 "word" 中的 'r' 换成 'o' ，得到 dictionary 中的单词 "wood" 。<br>- 将 "note" 中的 'n' 换成 'j' 且将 't' 换成 'k' ，得到 "joke" 。<br>- "ants" 需要超过 2 次编辑才能得到 dictionary 中的单词。<br>- "wood" 不需要修改（0 次编辑），就得到 dictionary 中相同的单词。<br>所以我们返回 ["word","note","wood"] 。<br></pre> <br><p><strong>示例 2：</strong></p> <br><pre><b>输入：</b>queries = ["yes"], dictionary = ["not"]<br><b>输出：</b>[]<br><strong>解释：</strong><br>"yes" 需要超过 2 次编辑才能得到 "not" 。<br>所以我们返回空数组。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= queries.length, dictionary.length <= 100</code></li><li><code>n == queries[i].length == dictionary[j].length</code></li><li><code>1 <= n <= 100</code></li><li>所有 <code>queries[i]</code> 和 <code>dictionary[j]</code> 都只包含小写英文字母。</li></ul> <br><h3><a id="_47"></a>解题方法：暴力匹配</h3>
