<h2><a id="LetMeFly1877_1"></a>【LetMeFly】1877.数组中最大数对和的最小值：排序</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/minimize-maximum-pair-sum-in-array/" rel="nofollow">https://leetcode.cn/problems/minimize-maximum-pair-sum-in-array/</a></p> <br><p>一个数对 <code>(a,b)</code> 的 <strong>数对和</strong> 等于 <code>a + b</code> 。<strong>最大数对和</strong> 是一个数对数组中最大的 <strong>数对和</strong> 。</p> <br><ul><li>比方说，如果我们有数对 <code>(1,5)</code> ，<code>(2,3)</code> 和 <code>(4,4)</code>，<strong>最大数对和</strong> 为 <code>max(1+5, 2+3, 4+4) = max(6, 5, 8) = 8</code> 。</li></ul> <br><p>给你一个长度为 <strong>偶数</strong> <code>n</code> 的数组 <code>nums</code> ，请你将 <code>nums</code> 中的元素分成 <code>n / 2</code> 个数对，使得：</p> <br><ul><li><code>nums</code> 中每个元素 <strong>恰好</strong> 在 <strong>一个</strong> 数对中，且</li><li><strong>最大数对和</strong> 的值 <strong>最小</strong> 。</li></ul> <br><p>请你在最优数对划分的方案下，返回最小的 <strong>最大数对和</strong> 。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><b>输入：</b>nums = [3,5,2,3]<br><b>输出：</b>7<br><b>解释：</b>数组中的元素可以分为数对 (3,3) 和 (5,2) 。<br>最大数对和为 max(3+3, 5+2) = max(6, 7) = 7 。<br></pre> <br><p><strong>示例 2：</strong></p> <br><pre><b>输入：</b>nums = [3,5,4,2,4,6]<br><b>输出：</b>8<br><b>解释：</b>数组中的元素可以分为数对 (3,5)，(4,4) 和 (6,2) 。<br>最大数对和为 max(3+5, 4+4, 6+2) = max(8, 8, 8) = 8 。<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>n == nums.length</code></li><li><code>2 <= n <= 10<sup>5</sup></code></li><li><code>n</code> 是
