rel="nofollow">https://leetcode.cn/problems/minimum-distance-between-three-equal-elements-i/</a></p> <br><p>给你一个整数数组 <code>nums</code>。</p> <br><p>如果满足 <code>nums[i] == nums[j] == nums[k]</code>，且 <code>(i, j, k)</code> 是 3 个 <strong>不同 </strong>下标，那么三元组 <code>(i, j, k)</code> 被称为 <strong>有效三元组 </strong>。</p> <br><p><strong>有效三元组 </strong>的 <strong>距离 </strong>被定义为 <code>abs(i - j) + abs(j - k) + abs(k - i)</code>，其中 <code>abs(x)</code> 表示 <code>x</code> 的 <strong>绝对值 </strong>。</p> <br><p>返回一个整数，表示 <strong>有效三元组 </strong>的 <strong>最小 </strong>可能距离。如果不存在 <strong>有效三元组 </strong>，返回 <code>nums[0] == nums[2] == nums[3] == 1</code>。它的距离为 <code>abs(0 - 2) + abs(2 - 3) + abs(3 - 0) = 2 + 1 + 3 = 6</code>。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">nums = [1,1,2,3,2,1,2]</span></p> <br></div><br><p><strong>输出：</strong> <span class="example-io">8</span></p> <br><p><strong>解释：</strong></p> <br><p>最小距离对应的有效三元组是 <code>(2, 4, 6)</code> 。</p> <br><p><code>(2, 4, 6)</code> 是一个有效三元组，因为 <code>nums[2] == nums[4] == nums[6] == 2</code>。它的距离为 <code>abs(2 - 4) + abs(4 - 6) + abs(6 - 2) = 2 + 2 + 4 = 8</code>。</p> <br><p><strong class="example">示例 3：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> <span class="example-io">nums =
