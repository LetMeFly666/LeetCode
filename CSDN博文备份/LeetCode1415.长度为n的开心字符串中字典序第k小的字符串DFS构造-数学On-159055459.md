<h2><a id="LetMeFly1415_n__k_DFS__On_1"></a>【LetMeFly】1415.长度为 n 的开心字符串中字典序第 k 小的字符串：DFS构造 / 数学O(n)</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/the-k-th-lexicographical-string-of-all-happy-strings-of-length-n/" rel="nofollow">https://leetcode.cn/problems/the-k-th-lexicographical-string-of-all-happy-strings-of-length-n/</a></p> <br><p>一个 「开心字符串」定义为：</p> <br><ul><li>仅包含小写字母 <code>['a', 'b', 'c']</code>.</li><li>对所有在 <code>1</code> 到 <code>s.length - 1</code> 之间的 <code>i</code> ，满足 <code>s[i] != s[i + 1]</code> （字符串的下标从 1 开始）。</li></ul> <br><p>比方说，字符串 <strong>"abc"</strong>，<strong>"ac"，"b"</strong> 和 <strong>"abcbabcbcb"</strong> 都是开心字符串，但是 <strong>"aa"</strong>，<strong>"baa"</strong> 和 <strong>"ababbc"</strong> 都不是开心字符串。</p> <br><p>给你两个整数 <code>n</code> 和 <code>k</code> ，你需要将长度为 <code>n</code> 的所有开心字符串按字典序排序。</p> <br><p>请你返回排序后的第 k 个开心字符串，如果长度为 <code>n</code> 的开心字符串少于 <code>k</code> 个，那么请你返回 <strong>空字符串</strong> 。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><strong>输入：</strong>n = 1, k = 3<br><strong>输出：</strong>"c"<br><strong>解释：</strong>列表 ["a", "b", "c"] 包含了所有长度为 1 的开心字符串。按照字典序排序后第三个字符串为 "c" 。<br></pre> <br><p><strong>示例 2：</strong></p> <br><pre><strong>输入：</strong>n = 1, k = 4<br><strong>输出：</strong>""<br><strong>解释：</strong>长度为 1 的开心字符串只有 3 个。<br></pre> <br><p><strong>示例 3：</strong></p> <br><pre><strong>输入：</strong>n = 3, k = 9<br><strong>输出：</strong>"cab"<br><strong>解释：</strong>长度为 3 的开心字符串总共有 12 个 ["aba", "abc", "aca", "acb", "bab", "bac", "bca", "bcb", "cab", "cac", "cba", "cbc"] 。第 9 个字符串为 "cab"<br></pre> <br><p><strong>示例 4：</strong></p> <br><pre><strong>输入：</strong>n = 2, k = 7<br><strong>输出：</strong>""<br></pre> <br><p><strong>示例 5：</strong></p> <br><pre><strong>输入：</strong>n = 10, k = 100<br><strong>输出：</strong>"abacbabacb"<br></pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= n <= 10</code></li><li><code>1 <= k <= 100</code></li></ul> <br><p> </p> <br><h3><a id="DFS_66"></a>解题方法一：DFS构造</h3> <br><p>写一个<code>dfs</code>函数接收已经构造的字符串并继续构造字符串：</p>
