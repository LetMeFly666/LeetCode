<h2><a id="LetMeFly3546_I_1"></a>【LetMeFly】3546.等和矩阵分割 I：记总记当前（遍历）</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/equal-sum-grid-partition-i/" rel="nofollow">https://leetcode.cn/problems/equal-sum-grid-partition-i/</a></p> <br><p>给你一个由正整数组成的 <code>m x n</code> 矩阵 <code>grid</code>。你的任务是判断是否可以通过 <strong>一条水平或一条垂直分割线 </strong>将矩阵分割成两部分，使得：</p> <br><ul><li>分割后形成的每个部分都是 <strong>非空 </strong>的。</li><li>两个部分中所有元素的和 <strong>相等 </strong>。</li></ul> <br><p>如果存在这样的分割，返回 <code>true</code>；否则，返回 <code>false</code>。</p> <br><p> </p> <br><p><strong class="example">示例 1：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> grid = [[1,4],[2,3]]</p> <br></div><br><p><strong>输出：</strong> true</p> <br><p><strong>解释：</strong></p> <br><p><img alt="" src="https://i-blog.csdnimg.cn/img_convert/ea23dc7b3213252ba1e6f10b90fcff2b.jpeg" /></p> <br><p>在第 0 行和第 1 行之间进行水平分割，得到两个非空部分，每部分的元素之和为 5。因此，答案是 <code>true</code>。</p> <br><p><strong class="example">示例 2：</strong></p> <br><div class="example-block"> <br> <p><strong>输入：</strong> grid = [[1,3],[2,4]]</p> <br></div><br><p><strong>输出：</strong> false</p> <br><p><strong>解释：</strong></p> <br><p>无论是水平分割还是垂直分割，都无法使两个非空部分的元素之和相等。因此，答案是 <code>false</code>。</p> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= m == grid.length <= 10<sup>5</sup></code></li><li><code>1 <= n == grid[i].length <= 10<sup>5</sup></code></li><li><code>2 <= m * n <= 10<sup>5</sup></code></li><li><code>1 <= grid[i][j] <= 10<sup>5</sup></code></li></ul> <br><p> </p> <br><h3><a id="_57"></a>解题方法：三次遍历</h3> <br><p>第一次遍历计算整个grid的元素之和，若非偶数直接返回<code>false</code>，若为偶数则直接除以<span class="katex--inline"><span class="katex"><span class="katex-mathml"><br>
