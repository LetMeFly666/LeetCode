<h2><a id="LetMeFly2506_1"></a>【LetMeFly】2506.统计相似字符串对的数目：哈希表+位运算</h2> <br><p>力扣题目链接：<a href="https://leetcode.cn/problems/count-pairs-of-similar-strings/" rel="nofollow">https://leetcode.cn/problems/count-pairs-of-similar-strings/</a></p> <br><p>给你一个下标从 <strong>0</strong> 开始的字符串数组 <code>words</code> 。</p> <br><p>如果两个字符串由相同的字符组成，则认为这两个字符串 <strong>相似</strong> 。</p> <br><ul><li>例如，<code>"abca"</code> 和 <code>"cba"</code> 相似，因为它们都由字符 <code>'a'</code>、<code>'b'</code>、<code>'c'</code> 组成。</li><li>然而，<code>"abacba"</code> 和 <code>"bcfd"</code> 不相似，因为它们不是相同字符组成的。</li></ul> <br><p>请你找出满足字符串 <code>words[i]</code><em> </em>和<em> </em><code>words[j]</code> 相似的下标对<em> </em><code>(i, j)</code><em> </em>，并返回下标对的数目，其中 <code>0 <= i < j <= words.length - 1</code> 。</p> <br><p> </p> <br><p><strong>示例 1：</strong></p> <br><pre><strong>输入：</strong>words = ["aba","aabb","abcd","bac","aabc"]<br><strong>输出：</strong>2<br><strong>解释：</strong>共有 2 对满足条件：<br>- i = 0 且 j = 1 ：words[0] 和 words[1] 只由字符 'a' 和 'b' 组成。 <br>- i = 3 且 j = 4 ：words[3] 和 words[4] 只由字符 'a'、'b' 和 'c' 。 <br></pre> <br><p><strong>示例 2：</strong></p> <br><pre><strong>输入：</strong>words = ["aabb","ab","ba"]<br><strong>输出：</strong>3<br><strong>解释：</strong>共有 3 对满足条件：<br>- i = 0 且 j = 1 ：words[0] 和 words[1] 只由字符 'a' 和 'b' 组成。 <br>- i = 0 且 j = 2 ：words[0] 和 words[2] 只由字符 'a' 和 'b' 组成。 <br>- i = 1 且 j = 2 ：words[1] 和 words[2] 只由字符 'a' 和 'b' 组成。 <br></pre> <br><p><strong>示例 3：</strong></p> <br><pre><strong>输入：</strong>words = ["nba","cba","dba"]<br><strong>输出：</strong>0<br><strong>解释：</strong>不存在满足条件的下标对，返回 0 。</pre> <br><p> </p> <br><p><strong>提示：</strong></p> <br><ul><li><code>1 <= words.length <= 100</code></li><li><code>1 <= words[i].length <= 100</code></li><li><code>words[i]</code> 仅由小写英文字母组成</li></ul> <br><h3><a id="_58"></a>解题方法：哈希表+位运算</h3> <br><p>这道题不考虑每个字符的出现次数，只考虑是否出现过。一共26个字母，因此可以使用一个整数二进制下的低26位分别代表每个字母是否出现过（可以将之称为<code>状态码</code>）。</p> <br><p>使用一个哈希表统计每种<code>状态码</code>的出现次数。遍历字符串数组，对于一个字符串，计算出它的<code>状态码</code>后：</p>
