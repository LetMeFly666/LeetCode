Windows - UWP - 通过链接启动Windows设置中的应用：ms-settings

前言

前段时间（大概是2023年2月23日）发现了浏览器可以打开Windows的“设置”中的“疑难解答”

通过“复制链接”的方式很容易得到其实是在浏览器中访问了

使用方法

在Windows10中，使用组合键Win + R，在“运行”中输入ms-settings:troubleshoot并回车即可打开Windows设置中的“疑难解答”

同样地，在浏览器（例如Google）的地址栏中输入ms-settings:troubleshoot并回车，浏览器会询问“是否打开”，点击打开即可

其中ms肯定是Microsoft的缩写，settings就是设置，troubleshoot就是疑难解答。

这样，就可以在自己的网页或程序中，打开对应的Windows设置了。

有没有更多的相关设置呢？当然有！
