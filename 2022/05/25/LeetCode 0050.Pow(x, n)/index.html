50.Pow(x, n)

【LetMeFly】50.Pow(x, n)

实现 pow(x, n) ，即计算x的n次幂函数（即，xⁿ ）。

提示:

  • −100.0<x<100.0-100.0 < x < 100.0−100.0<x<100.0
  • −231≤n≤231−1-2^{31} \leq n \leq 2^{31}-1−2​31​​≤n≤2​31​​−1
  • −104≤xn≤104-10^4 \leq x^n \leq 10^4−10​4​​≤x​n​​≤10​4​​

题目大意

给你一个浮点数xxx和一个整数nnn，让你计算xnx^nx​n​​。

思路

−104≤xn≤104-10^4 \leq x^n \leq 10^4−10​4​​≤x​n​​≤10​4​​，也就是说答案不会异常大。因此直接使用修改版的快速幂即可。

快速幂

快速幂的介绍有很多，比如可以参考 https://letmefly.xyz/Notes/ACM/Problems/快速幂 /

但是一般的快速幂都是计算整数的整数次幂的，并且会取模。

因此我们只需要做出一些特殊修改即可：

如果nnn是负数，就计算1/Pow(x,−n)1 / Pow(x, -n)1/Pow(x,−n)

注意：

一些编程语言中，nnn一定要记得转为long long，因为-INT_MIN > INT_MAX

方法一：快速幂

直接套用快速幂模板并稍作修改即可。

  • 时间复杂度O(logn)O(\log n)O(logn)
  • 空间复杂度O(1)O(1)O(1)
