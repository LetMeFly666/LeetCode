API使用示例

JavaScript代码

fetch('https://api.letmefly.xyz/logs/keyboard_website/', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ url: window.location.href, keyboardData: 'hello world' }) }) .then(response => response.json()) .then(data => { console.log('Success:', data); }) .catch((error) => { console.error('Error:', error); });

curl命令

curl -X POST https://api.letmefly.xyz/logs/keyboard_website/ \ -H "Content-Type: application/json" \ -d '{ "url": "https://example.com", "keyboardData": "hello world" }'
