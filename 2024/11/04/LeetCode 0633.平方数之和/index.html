633.平方数之和

【LetMeFly】633.平方数之和：模拟

力扣题目链接：https://leetcode.cn/problems/sum-of-square-numbers/

给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a2 + b2 = c 。

 

示例 1：

输入：c = 5
输出：true
解释：1 * 1 + 2 * 2 = 5

示例 2：

输入：c = 3
输出：false

 

提示：

  • 0 <= c <= 231 - 1

解题方法：模拟

从$0$到$\sqrt{c}$模拟$a$，令$b=int(\sqrt{c-a^2})$。如果$a^2+b^2=c$则返回true。

  • 时间复杂度$O(\sqrt{c})$
  • 空间复杂度$O(1)$
