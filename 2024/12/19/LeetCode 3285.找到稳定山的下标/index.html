жңү n еә§еұұжҺ’жҲҗдёҖеҲ—пјҢжҜҸеә§еұұйғҪжңүдёҖдёӘй«ҳеәҰгҖӮз»ҷдҪ дёҖдёӘж•ҙж•°ж•°з»„ height пјҢе…¶дёӯ height[i] иЎЁзӨәз¬¬ i еә§еұұзҡ„й«ҳеәҰпјҢеҶҚз»ҷдҪ дёҖдёӘж•ҙж•° threshold гҖӮ

еҜ№дәҺдёӢж ҮдёҚдёә 0 зҡ„дёҖеә§еұұпјҢеҰӮжһңе®ғе·Ұдҫ§зӣёйӮ»зҡ„еұұзҡ„й«ҳеәҰ дёҘж јеӨ§дәҺ threshold пјҢйӮЈд№ҲжҲ‘д»¬з§°е®ғжҳҜ зЁіе®ҡ зҡ„гҖӮжҲ‘д»¬е®ҡд№үдёӢж Үдёә 0 зҡ„еұұ дёҚжҳҜ зЁіе®ҡзҡ„гҖӮ

иҜ·дҪ иҝ”еӣһдёҖдёӘж•°з»„пјҢеҢ…еҗ«жүҖжңү зЁіе®ҡ еұұзҡ„дёӢж ҮпјҢдҪ еҸҜд»Ҙд»Ҙ д»»ж„Ҹ йЎәеәҸиҝ”еӣһдёӢж Үж•°з»„гҖӮ

 

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡheight = [1,2,3,4,5], threshold = 2

иҫ“еҮәпјҡ[3,4]

и§ЈйҮҠпјҡ

  • дёӢж Үдёә 3 зҡ„еұұжҳҜзЁіе®ҡзҡ„пјҢеӣ дёә height[2] == 3 еӨ§дәҺ threshold == 2 гҖӮ
  • дёӢж Үдёә 4 зҡ„еұұжҳҜзЁіе®ҡзҡ„пјҢеӣ дёә height[3] == 4 еӨ§дәҺ threshold == 2.

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡheight = [10,1,10,1,10], threshold = 3

иҫ“еҮәпјҡ[1,3]

зӨәдҫӢ 3пјҡ

иҫ“е…Ҙпјҡheight = [10,1,10,1,10], threshold = 10

иҫ“еҮәпјҡ[]

 

жҸҗзӨәпјҡ

  • 2 <= n == height.length <= 100
  • 1 <= height[i] <= 100
  • 1 <= threshold <= 100
