# 【LetMeFly】2883.删去丢失的数据

力扣题目链接：[https://leetcode.cn/problems/drop-missing-data/](https://leetcode.cn/problems/drop-missing-data/)

<pre>
DataFrame students
+-------------+--------+
| Column Name | Type   |
+-------------+--------+
| student_id  | int    |
| name        | object |
| age         | int    |
+-------------+--------+
</pre>

<p>在 <code>name</code> 列里有一些具有缺失值的行。</p>

<p>编写一个解决方案，删除具有缺失值的行。</p>

<p>返回结果格式如下示例所示。</p>

<p>&nbsp;</p>

<p><b>示例 1:</b></p>

<pre>
<strong>输入：
</strong>+------------+-------+-----+
| student_id | name  | age |
+------------+-------+-----+
| 32         | Piper | 5   |
| 217        | Grace | 19  |
| 779        | None  | 20  |
| 849        | None  | 14  |
+------------+-------+-----+
<strong>输出：
</strong>+------------+-------+-----+
| student_id | name  | age |
+------------+-------+-----+
| 32         | Piper | 5   |
| 217        | Grace | 19  |
+------------+-------+-----+
<b>解释：
</b>学号为 779 和 849 的学生所在行在 name 列中有空值，因此它们将被删除。</pre>
