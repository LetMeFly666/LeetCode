# 【LetMeFly】569.员工薪水中位数

力扣题目链接：[https://leetcode.cn/problems/median-employee-salary/](https://leetcode.cn/problems/median-employee-salary/)

<p>表:&nbsp;<code>Employee</code></p>

<pre>
+--------------+---------+
| Column Name  | Type    |
+--------------+---------+
| id           | int     |
| company      | varchar |
| salary       | int     |
+--------------+---------+
Id是该表的主键列。
该表的每一行表示公司和一名员工的工资。
</pre>

<p>&nbsp;</p>

<p>写一个SQL查询，找出每个公司的工资中位数。</p>

<p>以 <strong>任意顺序</strong> 返回结果表。</p>
