гҖҗLetMeFlyгҖ‘3176.жұӮеҮәжңҖй•ҝеҘҪеӯҗеәҸеҲ— IпјҡеҠЁжҖҒи§„еҲ’пјҲDPпјү

еҠӣжүЈйўҳзӣ®й“ҫжҺҘпјҡhttps://leetcode.cn/problems/find-the-maximum-length-of-a-good-subsequence-i/

з»ҷдҪ дёҖдёӘж•ҙж•°ж•°з»„ nums е’ҢдёҖдёӘ йқһиҙҹ ж•ҙж•° k гҖӮеҰӮжһңдёҖдёӘж•ҙж•°еәҸеҲ— seq ж»Ўи¶іеңЁиҢғеӣҙдёӢж ҮиҢғеӣҙ [0, seq.length - 2] дёӯеӯҳеңЁ дёҚи¶…иҝҮ k дёӘдёӢж Ү i ж»Ўи¶і seq[i] != seq[i + 1] пјҢйӮЈд№ҲжҲ‘д»¬з§°иҝҷдёӘж•ҙж•°еәҸеҲ—дёә еҘҪ еәҸеҲ—гҖӮ

иҜ·дҪ иҝ”еӣһ nums дёӯ еҘҪ еӯҗеәҸеҲ— зҡ„жңҖй•ҝй•ҝеәҰ

 

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡnums = [1,2,1,1,3], k = 2

иҫ“еҮәпјҡ4

и§ЈйҮҠпјҡ

жңҖй•ҝеҘҪеӯҗеәҸеҲ—дёә [1,2,1,1,3] гҖӮ

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡnums = [1,2,3,4,5,1], k = 0

иҫ“еҮәпјҡ2

и§ЈйҮҠпјҡ

жңҖй•ҝеҘҪеӯҗеәҸеҲ—дёә [1,2,3,4,5,1] гҖӮ

 

жҸҗзӨәпјҡ

  • 1 <= nums.length <= 500
  • 1 <= nums[i] <= 109
  • 0 <= k <= min(nums.length, 25)

и§Јйўҳж–№жі•пјҡеҠЁжҖҒи§„еҲ’

дҪҝз”ЁдёҖдёӘеҠЁжҖҒи§„еҲ’ж•°з»„$dp$пјҢе…¶дёӯ$dp[i][l]$д»ЈиЎЁж•°з»„еүҚ$i$дёӘе…ғзҙ зҡ„дёҚи¶…иҝҮ$l$дёӘзӣёйӮ»дёҚеҗҢзҡ„еҘҪеӯҗеәҸеҲ—зҡ„жңҖеӨ§й•ҝеәҰгҖӮ

еҲқе§ӢеҖј$dp[i][0]=1$пјҢе…¶дҪҷеҖјй»ҳи®Өдёә$-1$е°ұиЎҢпјҢиҪ¬з§»ж–№зЁӢпјҡ

$$dp[i][l] = \min \begin{cases}
dp[i][l] & \
dp[j][l] + 1 & \text{ if } nums[i]=nums[j]\
dp[j][l - 1] + 1 & \text{ if } nums[i] \neq nums[j] \text{ and } l \gt 0
\end{cases}$$

дёүеұӮеҫӘзҺҜпјҢз¬¬дёҖеұӮз”Ё$i$д»Һ$0$еҲ°$len(nums)-1$пјҢз¬¬дәҢеұӮз”Ё$l$д»Һ$0$еҲ°$k$пјҢз¬¬дёүеұӮз”Ё$j$д»Һ$0$еҲ°$i-1$гҖӮ

  • ж—¶й—ҙеӨҚжқӮеәҰ$O(len(nums)^2\times k)$
  • з©әй—ҙеӨҚжқӮеәҰ$O(len(nums)\times k)$
