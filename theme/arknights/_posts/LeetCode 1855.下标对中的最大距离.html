1855.дёӢж ҮеҜ№дёӯзҡ„жңҖеӨ§и·қзҰ»пјҡеҸҢжҢҮй’Ҳ

ж–Үз« еҸ‘еёғж—¶й—ҙ:
2026-04-19

гҖҗLetMeFlyгҖ‘1855.дёӢж ҮеҜ№дёӯзҡ„жңҖеӨ§и·қзҰ»пјҡеҸҢжҢҮй’Ҳ

еҠӣжүЈйўҳзӣ®й“ҫжҺҘпјҡhttps://leetcode.cn/problems/maximum-distance-between-a-pair-of-values/

з»ҷдҪ дёӨдёӘ йқһйҖ’еўһ зҡ„ж•ҙж•°ж•°з»„ nums1вҖӢвҖӢвҖӢвҖӢвҖӢвҖӢ е’Ң nums2вҖӢвҖӢвҖӢвҖӢвҖӢвҖӢ пјҢж•°з»„дёӢж ҮеқҮ д»Һ 0 ејҖе§Ӣ и®Ўж•°гҖӮ

дёӢж ҮеҜ№ (i, j) дёӯ 0 <= i < nums1.length дё” 0 <= j < nums2.length гҖӮеҰӮжһңиҜҘдёӢж ҮеҜ№еҗҢж—¶ж»Ўи¶і i <= j дё” nums1[i] <= nums2[j] пјҢеҲҷз§°д№Ӣдёә жңүж•Ҳ дёӢж ҮеҜ№пјҢиҜҘдёӢж ҮеҜ№зҡ„ и·қзҰ» дёә j - iвҖӢвҖӢ гҖӮвҖӢвҖӢ

иҝ”еӣһжүҖжңү жңүж•Ҳ дёӢж ҮеҜ№ (i, j) дёӯзҡ„ жңҖеӨ§и·қзҰ» гҖӮеҰӮжһңдёҚеӯҳеңЁжңүж•ҲдёӢж ҮеҜ№пјҢиҝ”еӣһ 0 гҖӮ

дёҖдёӘж•°з»„ arr пјҢеҰӮжһңжҜҸдёӘ 1 <= i < arr.length еқҮжңү arr[i-1] >= arr[i] жҲҗз«ӢпјҢйӮЈд№ҲиҜҘж•°з»„жҳҜдёҖдёӘ йқһйҖ’еўһ ж•°з»„гҖӮ

 

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡnums1 = [55,30,5,4,2], nums2 = [100,20,10,10,5]
иҫ“еҮәпјҡ2
и§ЈйҮҠпјҡжңүж•ҲдёӢж ҮеҜ№жҳҜ (0,0), (2,2), (2,3), (2,4), (3,3), (3,4) е’Ң (4,4) гҖӮ
жңҖеӨ§и·қзҰ»жҳҜ 2 пјҢеҜ№еә”дёӢж ҮеҜ№ (2,4) гҖӮ

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡnums1 = [2,2,2], nums2 = [10,10,1]
иҫ“еҮәпјҡ1
и§ЈйҮҠпјҡжңүж•ҲдёӢж ҮеҜ№жҳҜ (0,0), (0,1) е’Ң (1,1) гҖӮ
жңҖеӨ§и·қзҰ»жҳҜ 1 пјҢеҜ№еә”дёӢж ҮеҜ№ (0,1) гҖӮ

зӨәдҫӢ 3пјҡ

иҫ“е…Ҙпјҡnums1 = [30,29,19,5], nums2 = [25,25,25,25,25]
иҫ“еҮәпјҡ2
и§ЈйҮҠпјҡжңүж•ҲдёӢж ҮеҜ№жҳҜ (2,2), (2,3), (2,4), (3,3) е’Ң (3,4) гҖӮ
жңҖеӨ§и·қзҰ»жҳҜ 2 пјҢеҜ№еә”дёӢж ҮеҜ№ (2,4) гҖӮ

 

жҸҗзӨәпјҡ

  • 1 <= nums1.length <= 105
  • 1 <= nums2.length <= 105
  • 1 <= nums1[i], nums2[j] <= 105
  • nums1 е’Ң nums2 йғҪжҳҜ йқһйҖ’еўһ ж•°з»„

и§Јйўҳж–№жі•пјҡеҸҢжҢҮй’Ҳ

еҸҜд»Ҙжһҡдёҫ$j$пјҢйҡҸзқҖ$j$зҡ„еўһеӨ§пјҢ$nums2[j]$еҸҳе°ҸпјҢж»Ўи¶і$nums1[i]\leq nums2[j]$зҡ„$nums1[j]$дјҡеҸҳе°ҸпјҢ$i$дјҡеўһеӨ§гҖӮ

дҪҝз”ЁеҸҢжҢҮй’ҲпјҢеңЁжһҡдёҫ$j$зҡ„ж—¶еҖҷпјҢдёҚж–ӯеўһеӨ§$i$е№¶дҝқжҢҒ$nums1[i]\leq nums2[j]$гҖӮеҰӮжһң$i$и¶…еҮәж•°з»„иҢғеӣҙпјҢеҲҷиҜҙжҳҺ$nums1$дёӯжңҖе°Ҹзҡ„е…ғзҙ д»Қз„¶еӨ§дәҺ$nums2[j]$пјҢйҖҖеҮәгҖӮ

  • ж—¶й—ҙеӨҚжқӮеәҰ$O(len(nums1)+len(nums2))$
  • з©әй—ҙеӨҚжқӮеәҰ$O(1)$
