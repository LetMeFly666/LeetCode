з»ҷдҪ дёҖдёӘж•ҙж•° nгҖӮ

е®ҡд№үе®ғзҡ„ й•ңеғҸи·қзҰ» дёәпјҡabs(n - reverse(n))вҖӢвҖӢвҖӢвҖӢвҖӢвҖӢвҖӢпјҢе…¶дёӯ reverse(n) иЎЁзӨәе°Ҷ n зҡ„ж•°еӯ—еҸҚиҪ¬еҗҺеҪўжҲҗзҡ„ж•ҙж•°гҖӮ

иҝ”еӣһиЎЁзӨә n зҡ„й•ңеғҸи·қзҰ»зҡ„ж•ҙж•°гҖӮ

е…¶дёӯпјҢabs(x) иЎЁзӨә x зҡ„з»қеҜ№еҖјгҖӮ

 

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡ n = 25

иҫ“еҮәпјҡ 27

и§ЈйҮҠпјҡ

  • reverse(25) = 52гҖӮ
  • еӣ жӯӨпјҢзӯ”жЎҲдёә abs(25 - 52) = 27гҖӮ

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡ n = 10

иҫ“еҮәпјҡ 9

и§ЈйҮҠпјҡ

  • reverse(10) = 01пјҢеҚі 1гҖӮ
  • еӣ жӯӨпјҢзӯ”жЎҲдёә abs(10 - 1) = 9гҖӮ

зӨәдҫӢ 3пјҡ

иҫ“е…Ҙпјҡ n = 7

иҫ“еҮәпјҡ 0

и§ЈйҮҠпјҡ

  • reverse(7) = 7гҖӮ
  • еӣ жӯӨпјҢзӯ”жЎҲдёә abs(7 - 7) = 0гҖӮ

 

жҸҗзӨәпјҡ

  • 1 <= n <= 109
