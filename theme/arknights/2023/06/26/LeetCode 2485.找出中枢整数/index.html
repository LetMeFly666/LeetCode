з»ҷдҪ дёҖдёӘжӯЈж•ҙж•° n пјҢжүҫеҮәж»Ўи¶ідёӢиҝ°жқЎд»¶зҡ„ дёӯжһўж•ҙж•° x пјҡ

  • 1 е’Ң x д№Ӣй—ҙзҡ„жүҖжңүе…ғзҙ д№Ӣе’ҢзӯүдәҺ x е’Ң n д№Ӣй—ҙжүҖжңүе…ғзҙ д№Ӣе’ҢгҖӮ

иҝ”еӣһдёӯжһўж•ҙж•° x гҖӮеҰӮжһңдёҚеӯҳеңЁдёӯжһўж•ҙж•°пјҢеҲҷиҝ”еӣһ -1 гҖӮйўҳзӣ®дҝқиҜҒеҜ№дәҺз»ҷе®ҡзҡ„иҫ“е…ҘпјҢиҮіеӨҡеӯҳеңЁдёҖдёӘдёӯжһўж•ҙж•°гҖӮ

 

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡn = 8
иҫ“еҮәпјҡ6
и§ЈйҮҠпјҡ6 жҳҜдёӯжһўж•ҙж•°пјҢеӣ дёә 1 + 2 + 3 + 4 + 5 + 6 = 6 + 7 + 8 = 21 гҖӮ

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡn = 1
иҫ“еҮәпјҡ1
и§ЈйҮҠпјҡ1 жҳҜдёӯжһўж•ҙж•°пјҢеӣ дёә 1 = 1 гҖӮ

зӨәдҫӢ 3пјҡ

иҫ“е…Ҙпјҡn = 4
иҫ“еҮәпјҡ-1
и§ЈйҮҠпјҡеҸҜд»ҘиҜҒжҳҺдёҚеӯҳеңЁж»Ўи¶ійўҳзӣ®иҰҒжұӮзҡ„ж•ҙж•°гҖӮ

 

жҸҗзӨәпјҡ

  • 1 <= n <= 1000

ж–№жі•дёҖпјҡж•°еӯҰ

еҰӮжһңвҖң1 е’Ң x д№Ӣй—ҙзҡ„жүҖжңүе…ғзҙ д№Ӣе’ҢзӯүдәҺ x е’Ң n д№Ӣй—ҙжүҖжңүе…ғзҙ д№Ӣе’ҢвҖқпјҢ

йӮЈд№Ҳжңүпјҡ

$$1 + 2 + 3 + вҖҰ + x = x + (x + 1) + вҖҰ + n$$

дәҺжҳҜжңүпјҡ

$$\frac{x * (x + 1)}{2} = \frac{(n - x + 1) * (x + n)}{2}$$

и§Јеҫ—пјҡ

$$x = \sqrt{\frac{n^2 + n}{2}}$$

еӣ дёә$n^2 + n=n(n+1)$дёҖе®ҡжҳҜеҒ¶ж•°пјҢжүҖд»Ҙе…¶дёҖе®ҡиғҪж•ҙйҷӨ$2$гҖӮ

жҲ‘д»¬еҸӘйңҖиҰҒеҲӨж–ӯдёҖдёӢ$\frac{n^2 + n}{2}$жҳҜеҗҰжҳҜе№іж–№ж•°е°ұеҘҪдәҶ

  • ж—¶й—ҙеӨҚжқӮеәҰ$O(1)$
  • з©әй—ҙеӨҚжқӮеәҰ$O(1)$
