918.зҺҜеҪўеӯҗж•°з»„зҡ„жңҖеӨ§е’ҢпјҡеҠЁжҖҒи§„еҲ’ + жҖқз»ҙпјҲеҸҚиҪ¬пјү

ж–Үз« еҸ‘еёғж—¶й—ҙ:
2023-07-20

гҖҗLetMeFlyгҖ‘918.зҺҜеҪўеӯҗж•°з»„зҡ„жңҖеӨ§е’ҢпјҡеҠЁжҖҒи§„еҲ’ + жҖқз»ҙпјҲеҸҚиҪ¬пјү

еҠӣжүЈйўҳзӣ®й“ҫжҺҘпјҡhttps://leetcode.cn/problems/maximum-sum-circular-subarray/

з»ҷе®ҡдёҖдёӘй•ҝеәҰдёә n зҡ„зҺҜеҪўж•ҙж•°ж•°з»„ nums пјҢиҝ”еӣһ nums зҡ„йқһз©ә еӯҗж•°з»„ зҡ„жңҖеӨ§еҸҜиғҪе’Ң гҖӮ

зҺҜеҪўж•°з»„ ж„Ҹе‘ізқҖж•°з»„зҡ„жң«з«Ҝе°ҶдјҡдёҺејҖеӨҙзӣёиҝһе‘ҲзҺҜзҠ¶гҖӮеҪўејҸдёҠпјҢ nums[i] зҡ„дёӢдёҖдёӘе…ғзҙ жҳҜ nums[(i + 1) % n] пјҢ nums[i] зҡ„еүҚдёҖдёӘе…ғзҙ жҳҜ nums[(i - 1 + n) % n] гҖӮ

еӯҗж•°з»„ жңҖеӨҡеҸӘиғҪеҢ…еҗ«еӣәе®ҡзј“еҶІеҢә nums дёӯзҡ„жҜҸдёӘе…ғзҙ дёҖж¬ЎгҖӮеҪўејҸдёҠпјҢеҜ№дәҺеӯҗж•°з»„ nums[i], nums[i + 1], ..., nums[j] пјҢдёҚеӯҳеңЁ i <= k1, k2 <= j е…¶дёӯ k1 % n == k2 % n гҖӮ

 

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡnums = [1,-2,3,-2]
иҫ“еҮәпјҡ3
и§ЈйҮҠпјҡд»Һеӯҗж•°з»„ [3] еҫ—еҲ°жңҖеӨ§е’Ң 3

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡnums = [5,-3,5]
иҫ“еҮәпјҡ10
и§ЈйҮҠпјҡд»Һеӯҗж•°з»„ [5,5] еҫ—еҲ°жңҖеӨ§е’Ң 5 + 5 = 10

зӨәдҫӢ 3пјҡ

иҫ“е…Ҙпјҡnums = [3,-2,2,-3]
иҫ“еҮәпјҡ3
и§ЈйҮҠпјҡд»Һеӯҗж•°з»„ [3] е’Ң [3,-2,2] йғҪеҸҜд»Ҙеҫ—еҲ°жңҖеӨ§е’Ң 3

 

жҸҗзӨәпјҡ

  • n == nums.length
  • 1 <= n <= 3 * 104
  • -3 * 104 <= nums[i] <= 3 * 104вҖӢвҖӢвҖӢвҖӢвҖӢвҖӢвҖӢ

ж–№жі•дёҖпјҡеҠЁжҖҒи§„еҲ’ + жҖқз»ҙпјҲеҸҚиҪ¬пјү

йҰ–е…Ҳи®Ёи®әеҰӮдҪ•жұӮжҷ®йҖҡж•°з»„жңҖеӨ§еӯҗж•°з»„

еҫҲз®ҖеҚ•пјҢдҪҝз”Ё$i$йҒҚеҺҶж•°з»„пјҢдҪҝз”ЁдёҖдёӘеҸҳйҮҸ$nowM$з”ЁжқҘи®°еҪ•д»Ҙnums[i]з»“е°ҫзҡ„жңҖеӨ§еӯҗж•°з»„д№Ӣе’ҢпјҢдҪҝз”ЁеҸҰеӨ–дёҖдёӘеҸҳйҮҸ$M$жқҘи®°еҪ•ж•ҙдёӘиҝҮзЁӢдёӯзҡ„жңҖдјҳи§ЈгҖӮ

жҺҘзқҖи®Ёи®әеҰӮдҪ•д»Һжҷ®йҖҡж•°з»„еҲ°еҫӘзҺҜж•°з»„

е…¶е®һжҲ‘д»¬жҚўдёӘжҖқи·ҜпјҢиӢҘжҳҜдҪҝз”ЁдәҶеҫӘзҺҜж•°з»„зҡ„зү№жҖ§пјҲ$ж•°з»„еҗҺеҮ дёӘе…ғзҙ  + ж•°з»„еүҚеҮ дёӘе…ғзҙ $пјүпјҢдёҚе°ұзӯүд»·дәҺ$жҖ»ж•°з»„ - дёӯй—ҙеҮ дёӘе…ғзҙ $еҗ—пјҹ

еӣ жӯӨжҲ‘д»¬еҸӘйңҖиҰҒе’ҢвҖңи®Ўз®—жңҖеӨ§еӯҗж•°з»„вҖқзҡ„ж–№ејҸзӣёеҗҢпјҢи®Ўз®—еҮәвҖңжңҖе°Ҹеӯҗж•°з»„вҖқпјҢеҶҚдҪҝз”Ёж•°з»„жҖ»е’ҢеҮҸеҺ»жңҖе°Ҹеӯҗж•°з»„пјҢе°ұеҫ—еҲ°вҖңи·ЁдёӨз«Ҝзҡ„еӯҗж•°з»„вҖқзҡ„жңҖеӨ§е’ҢдәҶгҖӮ

з»ҶиҠӮжіЁж„Ҹ

жіЁж„Ҹеӯҗж•°з»„еҝ…йЎ»йқһз©әпјҢеӣ жӯӨеҰӮжһңж•°з»„дёӯжүҖжңүе…ғзҙ йғҪе°ҸдәҺ0зҡ„иҜқпјҲM < 0пјүпјҢдёҚиғҪиҝ”еӣһ0пјҲдёҖдёӘе…ғзҙ йғҪдёҚйҖүпјү

  • ж—¶й—ҙеӨҚжқӮеәҰ$O(len(nums))$
  • з©әй—ҙеӨҚжқӮеәҰ$O(1)$
